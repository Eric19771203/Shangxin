# 小红书（xhs）技能 v1.2.1
## 功能描述
小红书内容创作、发布、管理工具，支持爆款文案生成、标签优化、发布调度。

## 激活条件
用户提及"小红书"、"xhs"、"笔记"、"种草"等关键词时自动激活。

## 核心功能
1. **爆款文案生成**：根据主题生成符合小红书风格的笔记内容
2. **标签优化**：自动生成高流量标签组合
3. **发布规划**：制定发布时间表和内容策略
4. **数据分析**：分析笔记数据和热点趋势

## 使用格式
```
# 生成小红书笔记
skill:xhs:generate
主题：[你的主题]
风格：[可爱/干货/测评/种草]
字数：[300/500/800]
标签数量：[5/10/15]
```

## 输出规范
- 标题使用emoji开头，吸引人眼球
- 内容分段清晰，使用表情符号分隔
- 标签放在最后，包含热门标签+精准标签
- 符合小红书用户阅读习惯
