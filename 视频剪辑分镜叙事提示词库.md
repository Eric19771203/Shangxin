### 🎬 一、分镜/镜头提示词（英文+中文）
#### 景别（Shot Size）
- Extreme Wide Shot (EWS) / 极远景：展现宏大环境
- Wide Shot (WS) / 全景：人物全身+环境
- Medium Shot (MS) / 中景：腰部以上
- Medium Close-Up (MCU) / 中近景：胸部以上
- Close-Up (CU) / 特写：面部/局部
- Extreme Close-Up (ECU) / 极特写：细节（眼、手、道具）

#### 运镜（Camera Movement）
- Dolly in/out / 推/拉：前后移动
- Pan left/right / 摇：水平旋转
- Tilt up/down / 俯仰：垂直旋转
- Track / 跟拍：跟随主体移动
- Arc / 环绕：360°弧形运动
- Whip pan / 甩镜：快速旋转
- Vertigo zoom / 希区柯克变焦：推拉+反向变焦
- Handheld / 手持：轻微抖动，纪实感
- Gimbal / 稳定器：平滑运动
- Drone shot / 航拍：高空视角

#### 角度（Angle）
- Eye level / 平视：客观自然
- Low angle / 仰拍：强势、压迫
- High angle / 俯拍：渺小、无力
- Dutch angle / 荷兰角：倾斜，紧张/迷幻
- Over-the-shoulder (OTS) / 过肩：对话/对峙
- POV / 第一视角：沉浸式

#### 构图/镜头技巧
- Shallow depth of field / 浅景深：突出主体
- Rule of thirds / 三分法：平衡构图
- Leading lines / 引导线：引导视线
- Framing / 框架构图（门窗/拱门）
- Match cut / 匹配剪辑：动作/构图/颜色衔接
- Jump cut / 跳切：快速节奏
- Cross-cut / 交叉剪辑：并行叙事

### 📖 二、叙事/剪辑结构提示词（可直接套用）
#### 三幕剧剪辑模板（英文+中文）
```
### Three-Act Editing Prompt
Act 1 (Setup, 25%):
- Slow pacing, wide shots, character intro
- Inciting incident at 25% mark
- Hold 3s static shot before/after key beats

Act 2 (Confrontation, 50%):
- Rising tension, faster cuts, medium/close-ups
- Mini-climax every 10s
- Cross-cut parallel action

Act 3 (Resolution, 25%):
- Quick cuts to climax, slow fade to resolution
- Final hold on emotional close-up
- 5s silence/black at end
```
```
### 三幕剧剪辑提示词
第一幕（建置，25%）：
- 慢节奏、全景、人物出场
- 激励事件在25%处
- 关键节点前后各3秒静态镜头

第二幕（对抗，50%）：
- 节奏加快、中近景为主
- 每10秒一个小高潮
- 交叉剪辑并行剧情

第三幕（解决，25%）：
- 高潮快速剪辑，结尾渐缓
- 最后定格情感特写
- 结尾5秒留白/黑屏
```

Conflict (5–30s): Build tension, raise stakes
Climax (30s–2min): Main payoff, transformation
Resolution (last 30s): Emotional wrap-up

- Fast cuts between close-ups and wide shots
- Impact frames with motion blur
- Dynamic camera movement: tracking, dolly, whip pan

Style: action anime, cinematic fight, high energy, smooth animation.
