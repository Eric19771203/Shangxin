# Skill自动注册工具

## 🎯 功能说明

当你将新的skill文件移动到本目录后，使用这个工具可以自动：
1. 扫描skills/目录下的所有.md文件
2. 分析每个skill的功能和特点
3. 自动生成注册信息
4. 更新注册表

---

## 📋 使用方法

### 方式1：手动注册（推荐）

当你移动新的skill文件到skills/目录后，请提供以下信息：

```
@注册工具 注册新skill

文件名：[文件名].md
主要功能：[简要描述，1-2句话]
关键词：[关键词1, 关键词2, 关键词3]（用于匹配调用）
类型：[agent-based / function-based / prompt-based / tool-based]

输入要求：
- 必需：[字段1, 字段2]
- 可选：[字段3, 字段4]

输出内容：
- [输出1]
- [输出2]

是否需要Agent：[是/否]
如果需要，Agent名称：[名称]
```

### 方式2：批量扫描

```
@注册工具 扫描skills目录

我将自动扫描skills/目录下的所有.md文件，
并为每个文件生成注册信息供你确认。
```

### 方式3：智能分析

```
@注册工具 分析skill [文件名]

我将读取该skill文件的内容，
智能分析其功能、输入输出、关键词等，
自动生成注册信息。
```

## 🔧 注册信息模板

### 基础信息
```json
{
  "id": "NODE-XXX",  // 自动生成
  "name": "[节点名称]",
  "path": "skills/[文件名].md",
  "type": "[类型]",
  "description": "[描述]"
}
```

### 能力标签
```json
{
  "capabilities": [
    "[能力1]",
    "[能力2]",
    "[能力3]"
  ],
  "keywords": [
    "[关键词1]",
    "[关键词2]",
    "[关键词3]"
  ]
}
```

### 输入输出
```json
{
  "input": {
    "required": ["[字段1]", "[字段2]"],
    "optional": ["[字段3]"],
    "format": "text/json/yaml"
  },
  "output": {
    "format": "[格式]",
    "fields": ["[字段1]", "[字段2]"]
  }
}
```

### Agent配置（如需要）
```json
{
  "agent": "[agent-id]",
  "agent_config": {
    "name": "[Agent名称]",
    "skill_path": "skills/[文件名].md",
    "tools": ["readFile", "fsWrite"],
    "system_prompt": "[Prompt模板]"
  }
}
```

## 🚀 快速注册示例

### 示例1：创意生成类skill

```
@注册工具 注册新skill

文件名：创意生成器.md
主要功能：根据主题和风格生成短视频创意方案
关键词：创意, 生成, 头脑风暴, 策划
类型：agent-based

输入要求：
- 必需：topic（主题）, style（风格）
- 可选：duration（时长）, constraints（限制条件）

输出内容：
- 创意标题
- 核心概念
- 关键要点
- 执行建议

是否需要Agent：是
Agent名称：creative-generation-agent
```

**生成的注册信息**：
```json
{
  "id": "NODE-002",
  "name": "创意生成节点",
  "path": "skills/创意生成器.md",
  "type": "agent-based",
  "agent": "creative-generation-agent",
  "description": "根据主题和风格生成短视频创意方案",
  "capabilities": [
    "创意生成",
    "头脑风暴",
    "方案策划"
  ],
  "keywords": [
    "创意",
    "生成",
    "头脑风暴",
    "策划"
  ],
  "input": {
    "required": ["topic", "style"],
    "optional": ["duration", "constraints"],
    "format": "text"
  },
  "output": {
    "format": "creative-plan",
    "fields": ["title", "concept", "key_points", "suggestions"]
  }
}
```

## 🔄 更新注册表

注册完成后，我会自动更新以下文件：
1. `integrations/node-registry.json` - 添加新节点
2. `agents/agent-registry.json` - 添加新Agent（如需要）
3. `workflows/workflow-registry.json` - 更新可用节点列表

---

## 📝 注意事项

1. **节点ID自动生成**：按照NODE-001, NODE-002...顺序
2. **Agent ID命名规范**：小写字母+连字符，如content-review-agent
3. **关键词选择**：选择用户最可能使用的词汇
4. **类型判断**：
   - 需要复杂推理 → agent-based
   - 简单处理 → function-based
   - 只是模板 → prompt-based
   - 调用工具 → tool-based
