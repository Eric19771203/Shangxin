# 创意分析器 Skill

## 系统版本
短剧创意分析系统 V2.1

## 核心功能
对短剧创意进行多维度专业分析，评估市场潜力，提供优化建议。

## 输入要求

可接受以下任一形式：
1. 一句话故事梗概
2. 简单故事大纲
3. 核心创意概念
4. 已有IP改编想法

## 分析维度

### 1. 赛道定位

#### 主流赛道识别
- 霸总言情
- 逆袭爽文
- 悬疑推理
- 古装穿越
- 现代都市
- 玄幻仙侠
- 甜宠日常
- 复仇虐恋

#### 细分赛道
- 垂直领域（医疗、法律、金融等）
- 特殊设定（重生、系统、快穿等）
- 情感类型（暗恋、破镜重圆、先婚后爱等）

#### 赛道评估
- 市场热度：★★★★★
- 竞争程度：★★★★★
- 创新空间：★★★★★
- 制作难度：★★★★★

### 2. 受众评估

#### 目标用户画像
- **年龄层**：18-25岁/26-35岁/36-45岁/45岁以上
- **性别倾向**：女性向/男性向/全年龄向
- **消费能力**：高/中/低
- **观看场景**：碎片时间/深度追剧/社交分享

#### 受众匹配度
- 题材吸引力
- 情节共鸣度
- 角色认同感
- 付费意愿

#### 市场规模预估
- 潜在受众数量
- 付费转化率预期
- 传播潜力评估

### 3. 爽点解构

#### 核心爽点识别
- **主爽点**：故事的核心吸引力
- **辅爽点**：增强体验的次要爽点
- **爽点密度**：每集/每分钟的爽点数量

#### 爽点类型分析
- **情感爽点**：虐恋、甜宠、复仇、救赎
- **身份爽点**：身份反转、隐藏身份、身份碾压
- **能力爽点**：逆袭、打脸、装逼、开挂
- **关系爽点**：三角恋、多角恋、禁忌之恋
- **视觉爽点**：美颜、服化道、场景、特效

#### 爽点分布
- 开场爽点（0-30秒）
- 持续爽点（贯穿全剧）
- 高潮爽点（关键节点）
- 付费爽点（付费墙前后）

#### 爽点强度评估
- 新鲜度：★★★★★
- 冲击力：★★★★★
- 持续性：★★★★★
- 传播性：★★★★★

### 4. 优化建议

#### 核心情节优化
- **问题识别**：当前情节的薄弱环节
- **优化方向**：具体的改进建议
- **参考案例**：成功案例借鉴
- **实施难度**：改动的复杂程度

#### 叙事结构优化
- **节奏调整**：加快/放慢特定段落
- **冲突强化**：增强戏剧张力
- **悬念设置**：提升追剧欲望
- **付费点设计**：优化商业转化

#### 特色塑造建议
- **差异化定位**：与同类作品的区别
- **记忆点打造**：让观众记住的元素
- **传播点设计**：适合社交媒体传播的内容
- **IP潜力挖掘**：长期价值开发

#### 风险规避建议
- **内容风险**：政策、审核、价值观
- **市场风险**：同质化、过时、小众
- **制作风险**：成本、技术、周期
- **商业风险**：付费、传播、ROI

## 使用场景
- 创意初期评估
- 项目立项决策
- 剧本优化调整
- 市场定位分析
- 投资价值评估

## 注意事项
1. 分析要客观中立
2. 建议要具体可行
3. 避免主观臆断
4. 结合市场数据
5. 考虑制作成本
