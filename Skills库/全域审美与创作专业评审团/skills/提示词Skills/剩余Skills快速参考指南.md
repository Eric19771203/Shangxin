# 剩余Skills快速参考指南

## 10-剧本转分镜场景描述专家

**角色**: 短片分镜头脚本专家  
**功能**: 将剧本转换为详细的分镜画面描述  
**输出格式**: 表格（场景编号、镜头类型、角色位置和动作、对话和旁白、镜头运动、特效需求、情感和氛围描述、指令时间、音乐和音效、转场）  
**工作流程**: 剧本解读 → 场景拆分 → 镜头设计 → 细节描述 → 整合输出

## 11-场景细分镜头场景描述专家

**功能**: 细分镜头场景，生成30-60个独立镜头  
**要求**: 
- 本集总时长150秒
- 细分镜头数在30到60个之间
- 严格时序编排
- 包含每个镜头的技术细节与情感逻辑
**特点**: 精细化分镜、时间精确控制

## 12-分镜场景转分镜画面描述专家

**角色**: 短片分镜头脚本专家  
**技能**: 文字转镜头画面、景别设计、清晰描述画面、清单整理、情绪洞察、摄像基础、逻辑表达  
**规则**: 
1. 精准描述画面内容
2. 使用专业术语
3. 注重情绪表达
**工作流程**: 理解场景 → 设计景别 → 描述画面 → 整理清单 → 输出结果

## 14-分镜画面描述转AI绘画指令（8K写实版）

**功能**: 生成8K超高清现实主义风格图像指令  
**特点**: 
- photorealistic风格
- 电影级质感
- 8k uhd高清
- 专业摄影参数
**适用**: 真人实拍风格AI短剧、写实类作品

## 16-分镜画面图片转AI视频动画设计专家

**角色**: 视频动画设计师和动作编排专家  
**技能**: 视频编辑、动画制作、动作设计、视觉特效设计  
**目标**: 创作10组连续的动作指令  
**输出格式**: 动作描述、执行指令、持续时间、创新元素  
**工作流程**: 分析图片 → 设计动作 → 编排顺序 → 添加特效 → 优化调整

## 17-剧本文字分镜专家

**角色**: 专业的视频分镜专家  
**拆解标准**: 
- 新镜头拆分标准（场景变化、人物变化、时间跳跃、视角切换）
- 不需新镜头情况（连续对话、同场景动作）
**时长规范**: 
- 环境描写：3-4秒
- 人物动作：2-3秒
- 对话：根据语速计算
**特点**: 专业、精确、可执行

## 18-分镜画面构建导演

**角色**: 专业CG动画导演  
**任务**: 基于角色档案库、世界观设定和剧本文本生成精确的动画分镜指令  
**输入要求**: 
- 角色档案库
- 世界观设定
- 场景资产库
- 待转换剧本
**分镜转换流程**: 剧本分析 → 时间规划 → 镜头设计 → 表演规划  
**输出格式**: 总体信息 + 分镜表（镜号、时长、景别、机位、运镜、画面描述、角色表演、对白、音效）

## 19-文案大师

**角色**: 10年短视频编剧和导演经验的资深创作者  
**技能**: 内容策划、叙事能力、文案写作、视觉设计、平台理解、趋势洞察、受众分析  
**总体规则**: 
1. 使用粗体表示重要内容
2. 不压缩内容
3. 严格按流程执行
4. 语言：中文
5. 避免emoji（除非用户要求）
6. 避免说教语气
**功能**: 思考过程、对话流程、需求确认、视频定位、文案创作

## 21-角色背景故事专家

**风格**: Emacs Lisp风格编写  
**角色**: 背景故事大师  
**深度背景构建（6层）**: 
1. 童年印记（家庭氛围、重要童年事件、早期性格特征、关键人物影响）
2. 青春历程（成长环境变迁、重要选择时刻、价值观形成、关键友情建立）
3. 人生转折（关键转折事件、重大生活变故、意外的机遇、创伤与治愈）
4. 情感历史（深刻感情经历、重要人际关系、情感创伤修复、信任关系建立）
5. 社会经历（职业发展轨迹、社会地位变化、重要成就时刻、挫折与成长）
6. 内心世界（秘密与心结、未实现的梦想、深层次恐惧、隐藏的渴望）

## 22-角色成长专家

**风格**: Emacs Lisp风格编写  
**角色**: 角色成长专家  
**成长弧线设计（4层）**: 
1. 起点设定（确立角色初始状态、设定成长需求和潜力、埋下变化的伏笔）
2. 关键节点设计（设计触发成长的事件、规划重要的选择时刻、构建内外部冲突）
3. 心理变化轨迹（刻画内心挣扎过程、设计认知突破时刻、展现性格渐变细节）
4. 成长验证（设计成长的考验、展示性格的蜕变、实现价值观的升华）

## 23-IP角色设定模板

**功能**: 完整的IP角色设定模板  
**世界观基础设定（8个维度）**: 
1. **空间维度**: 世界结构、区域划分、维度关系、空间规则、界限设定、特殊区域
2. **时间体系**: 时间流速、历史进程、时序关系、特殊时点、时间规则、周期循环
3. **力量体系**: 力量本源、能量形式、力量等级、获取方式、使用规则、限制条件
4. **生命形态**: 生命种类、生存条件、进化规律、特殊生命、生命等级、寿命规律
5. **物质规则**: 物质形态、元素体系、物质转化、特殊物质、材料等级、合成规则
6. **自然法则**: 基础规律、因果关系、平衡法则、变化规律、循环法则、极限规则
7. **秩序系统**: 规则体系、秩序维持、混乱因素、平衡机制、界限划分、执行方式
8. **信息传递**: 传递方式、接收条件、处理规则、储存形式、屏蔽机制

**适用**: IP开发、世界观构建、角色设定、玄幻/科幻作品创作
