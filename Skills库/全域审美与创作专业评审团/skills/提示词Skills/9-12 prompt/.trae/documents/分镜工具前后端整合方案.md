## 整合方案概述

将已有的 Python 分镜工具作为后端 API，React 前端作为 UI 界面，实现前后端分离架构。

## 后端 API 设计 (FastAPI)

### 1. 生成 API
```python
POST /api/generate
{
  "content_type": "movie",      // movie/short_drama/comic_drama/tvc
  "subtype": "war",             // 子类型
  "board_type": "3x3",          // 3x3/4x3
  "scene_description": "...",   // 场景描述
  "director_style": "nolan",    // 导演风格ID
  "custom_prompts": {...}       // 自定义提示词
}
```

### 2. 知识库 API
```python
GET /api/content-types          // 获取所有内容类型
GET /api/directors              // 获取所有导演
GET /api/books                  // 获取参考书籍
GET /api/theory/{topic}         // 获取理论知识
```

### 3. 文件上传 API
```python
POST /api/upload-book           // 上传参考书籍
POST /api/upload-image          // 上传分镜参考图
```

## 前端页面设计

### 页面1: 主工作台 (Workflow)
- 左侧：内容类型选择 (电影/短剧/漫剧/TVC)
- 中间：3×3/4×3 分镜板展示
- 右侧：AI 生成面板（场景描述、导演风格、参数调节）

### 页面2: 导演风格库
- 18位导演卡片展示
- 风格对比功能
- 应用到生成器

### 页面3: 知识库
- 参考书籍管理
- 理论知识查询
- 视觉术语词典

### 页面4: 项目导出
- 分镜导出 (PDF/图片)
- 提示词导出
- 项目保存/加载
