# 产品摄影专业知识库

## 概述

本知识库提供电商产品摄影的专业布光、拍摄和后期技巧，确保AI生成图像符合商业摄影标准。

---

## 基础布光方案

### 1. 三灯布光 (Three-Point Lighting)

**经典布光方案**:

```
        [Key Light] 主光 45°
              \
               \
                \ [产品]
                / 
               /  
        [Fill Light] 补光
        
        [Back Light/Rim] 轮廓光
```

**主光 (Key Light)**:
- 位置: 45度角，高于产品
- 作用: 主要照明，塑造立体感
- 强度: 100%（基准）
- 设备: 柔光箱、伞灯

**补光 (Fill Light)**:
- 位置: 另一侧，低于主光
- 作用: 填充阴影，降低对比
- 强度: 主光的50-75%
- 设备: 反光板、柔光箱

**轮廓光 (Rim Light)**:
- 位置: 后方，打亮边缘
- 作用: 分离背景，突出轮廓
- 强度: 主光的50-100%
- 效果: 白色边缘光（高端感）

---

### 2. 白底产品照 (White Background Product Shot)

**适用场景**:
- 电商主图
- 产品目录
- 后期抠图需求

**布光方案**:

```
[Left Softbox]      [Product]      [Right Softbox]
     8                     8              8
     |                     |              |
     \                     |             /
      \                    |            /
       \___________________|___________/
                   
        [Background Light - 均匀照亮背景]
```

**关键要点**:
1. 背景单独打光，纯白无影
2. 产品两侧柔光，对称照明
3. 顶部可选：柔化高光
4. 底部反光板：补亮底部阴影

**AI提示词**:
```
product photography on pure white background, soft box lighting, clean studio setup, no shadows, e-commerce style, professional commercial photography, high detail, sharp focus
```

---

### 3. 悬浮效果 (Floating Product)

**布光方案**:
- 顶部柔光箱（主光）
- 两侧条形柔光箱（勾勒轮廓）
- 底部反光板或柔光箱（补光）
- 黑色背景（产品悬浮感）

**技巧**:
- 实际拍摄可用透明支架
- 后期去除支架
- 阴影要自然（表明重量）

**AI提示词**:
```
floating product photography, levitating in mid-air, soft studio lighting, minimalist black background, gravity-defying, professional commercial, high-end luxury aesthetic
```

## 材质表现

### 4. 金属/反光材质

**挑战**:
- 容易反光
- 指纹明显
- 需要展示质感

**布光方案**:
- **大面积柔光源**: 柔光箱、柔光纸
- **避免直射**: 用旗板遮挡硬光
- **环境反射**: 控制反射内容（白色卡纸、渐变纸）
- **偏振镜**: 减少不必要的反光

**渐变照明**:
- 创造金属表面的渐变光泽
- 表现质感和高级感
- 常用：从深到浅或从亮到暗

**AI提示词**:
```
metal product photography, gradient lighting, soft reflection, brushed metal texture, premium finish, professional studio, soft box lighting, high-end commercial
```

---

### 5. 玻璃/透明材质

**挑战**:
- 边缘不易看见
- 反光强烈
- 需要展示透明度

**布光方案**:
- **背光照明**: 展示透明度和液体颜色
- **边缘光**: 勾勒轮廓（深色背景）
- **侧光**: 展示玻璃厚度
- **避免**: 正面直射（会反光）

**技巧**:
- 用黑色卡纸在两侧制造边缘线
- 背光可以用渐变板创造层次
- 灰尘要彻底清洁

**AI提示词**:
```
glass bottle product photography, backlit illumination, transparent liquid, rim lighting on dark background, showing clarity and refraction, professional studio, premium beverage
```

---

### 6. 食品/美食摄影

**布光方案**:
- **自然光优先**: 窗户光，柔和自然
- **侧光/侧逆光**: 表现纹理（面包、肉类）
- **补光**: 反光板填充阴影
- **避免**: 顶光（扁平无趣）

**增强质感的技巧**:
- **喷水**: 新鲜蔬菜水果
- **刷油**: 肉类光泽
- **蒸汽**: 热食效果（需后期或实拍）
- **手工造型**: 不完美中的完美

**AI提示词**:
```
food photography, natural window light, side lighting, steam rising, fresh ingredients, appetizing, shallow depth of field, rustic styling, professional culinary photography
```

---

### 7. 化妆品/护肤品

**布光方案**:
- **柔和均匀光**: 减少强烈阴影
- **渐变背景**: 从浅到深（高级感）
- **反射控制**: 瓶身标签清晰可见
- **微距镜头**: 展现质地

**质地展示**:
- 奶油质地：平滑流动
- 液体质地：滴落瞬间
- 粉末质地：细腻飞扬
- 凝胶质地：透明光泽

**AI提示词**:
```
cosmetics product photography, soft even lighting, gradient background, showing texture and consistency, premium skincare, glass bottle with golden cap, luxury aesthetic, professional beauty commercial
```

## 场景拍摄

### 8. 生活方式场景 (Lifestyle)

**原则**:
- 真实自然，不刻意
- 展示使用场景
- 营造氛围而非硬推产品

**布光**:
- 自然光为主
- 补光要无痕（看不出打光）
- 保持场景整体感

**构图**:
- 产品不一定要在中心
- 展示环境氛围
- 人物与产品互动自然

**AI提示词**:
```
lifestyle product photography, natural daylight, authentic scene, person using product in daily life, warm atmosphere, candid moment, professional commercial, cozy home setting
```

---

### 9. 纯色背景+道具

**选择背景色**:
- **白色**: 通用、干净、百搭
- **黑色**: 高端、神秘、科技感
- **灰色**: 中性、专业
- **彩色**: 品牌色、情绪色（红=热情、蓝=冷静）

**道具搭配**:
- 与产品相关的元素
- 不抢镜但增加层次
- 颜色协调
- 质感对比（粗糙vs光滑）

**AI提示词**:
```
product photography with props, pastel pink background, complementary styling elements, balanced composition, soft shadows, professional studio, minimalist aesthetic with texture
```

## 特殊效果

### 10. 水花/液体飞溅

**拍摄技巧**:
- 高速快门（1/2000s以上）
- 连拍模式
- 人工制造飞溅
- 多次拍摄选最佳

**布光**:
- 背光或侧逆光（液体透亮）
- 深色背景（水花清晰）
- 闪光灯凝固瞬间

**AI提示词**:
```
product photography with water splash, high speed capture, liquid droplets frozen in air, dynamic action, refreshing feeling, dark background, professional studio, refreshing beverage
```

---

### 11. 烟雾/粉末效果

**布光**:
- **背光**: 烟雾清晰可见
- **黑色背景**: 烟雾轮廓
- **侧光**: 烟雾层次

**AI提示词**:
```
product photography with smoke, mysterious atmosphere, backlit smoke, dark moody background, powder explosion, dynamic composition, professional studio, atmospheric lighting
```

---

### 12. 倒影效果

**镜面反射**:
- 使用黑色亚克力板（产品倒影清晰）
- 或白色亚克力（柔和倒影）
- 控制倒影清晰度（用布遮挡部分）

**水面效果**:
- 实际水面拍摄
- 或后期添加
- 增加产品高级感

**AI提示词**:
```
product photography with reflection, mirror surface, symmetrical composition, premium luxury aesthetic, water reflection, professional studio, elegant and sophisticated
```

## 色彩与调色

### 13. 色彩心理学应用

| 色彩 | 情感 | 适用产品 |
|------|------|----------|
| **白** | 纯净、医疗、简约 | 护肤品、医疗产品 |
| **黑** | 奢华、神秘、高端 | 奢侈品、科技产品 |
| **金** | 财富、品质、传统 | 保健品、高档食品 |
| **蓝** | 信任、科技、冷静 | 科技产品、金融 |
| **绿** | 自然、健康、环保 | 有机食品、天然产品 |
| **粉** | 女性、温柔、浪漫 | 美妆、女性产品 |
| **橙** | 活力、食欲、温暖 | 食品、运动产品 |

---

### 14. 后期调色LUT方向

**清新明亮**:
- 高亮度
- 低对比度
- 饱和度适中
- 偏冷色调
- 适用：护肤品、夏季产品

**温暖复古**:
- 暖色调
- 柔和高光
- 胶片颗粒
- 适用：食品、传统产品

**高级冷淡**:
- 低饱和度
- 对比度适中
- 偏青或偏蓝
- 适用：科技、奢侈品

**鲜艳活力**:
- 高饱和度
- 高对比度
- 色彩鲜明
- 适用：运动、零食、年轻品牌

## 相机设置参考

### 产品静物标准设置

**光圈**: f/8 - f/11（产品整体清晰）
**快门**: 1/125s（避免手抖，用三脚架）
**ISO**: 100-400（画质最佳）
**白平衡**: 手动设定或RAW后期调整
**对焦**: 单点对焦，产品中心或前1/3
**格式**: RAW（后期空间大）

### 动态产品设置

**光圈**: f/5.6 - f/8（平衡景深和进光）
**快门**: 1/500s以上（凝固动作）
**ISO**: 400-1600（必要时提高）
**对焦**: 连续自动对焦（AI Servo）
**连拍**: 开启

## AI生成图像优化建议

### 提示词结构优化

```
[产品类型] + [材质] + [布光方案] + [背景] + [质感形容词] + [商业摄影术语]

示例：
premium skincare serum bottle, frosted glass with gold accents, soft box three-point lighting, gradient pink background, luxurious texture, professional commercial product photography, 8k, sharp focus, studio quality
```

### 避免常见问题

- **阴影过重**: 添加"soft shadows"、"even lighting"
- **塑料感**: 添加"premium materials"、"high-quality finish"
- **不真实**: 添加"photorealistic"、"professional photography"
- **光线杂乱**: 指定"clean lighting"、"studio setup"
