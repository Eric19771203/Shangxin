# 专业分镜构图知识库

## 概述

本知识库提供电影级别的分镜构图理论与实战技巧，提升TVC的视觉专业度。

---

## 基础构图法则

### 1. 三分法则 (Rule of Thirds)

**原理**:
将画面分成九宫格，重要元素放在交叉点或线上。

**应用**:
- 人物眼睛放在上线
- 地平线放在上下线
- 产品焦点放在交叉点

**TVC示例**:
```
产品展示时，将产品主体放在画面右侧1/3处
左侧2/3留白或放置使用场景
创造视觉平衡和呼吸空间
```

**AI提示词增强**:
```
rule of thirds composition, off-center product placement, balanced negative space, professional framing
```

---

### 2. 黄金分割 (Golden Ratio)

**原理**:
1:1.618的完美比例，自然界普遍存在的和谐比例。

**视觉表现**:
- 黄金螺旋引导视线
- 斐波那契数列构图
- 比三分法更微妙的平衡

**TVC应用**:
- 奢侈品展示
- 艺术作品呈现
- 自然元素组合

**AI提示词**:
```
golden ratio composition, fibonacci spiral, divine proportion, harmonious balance
```

---

### 3. 对称构图 (Symmetrical Composition)

**类型**:
- **水平对称**: 倒影、建筑
- **垂直对称**: 人物、产品
- **径向对称**: 圆形图案、花朵

**心理效应**:
- 稳定、正式、庄严
- 秩序感、完美主义
- 仪式感

**TVC应用**:
- 高端产品（Wes Anderson风格）
- 建筑类广告
- 仪式感的开箱

**注意事项**:
- 避免完全对称的呆板
- 加入打破对称的元素（如一颗石子投入平静水面）

---

### 4. 引导线 (Leading Lines)

**类型**:
- **实际线条**: 道路、建筑线条、栏杆
- **虚拟线条**: 视线方向、动作轨迹
- **隐含线条**: 光影、色彩对比

**作用**:
- 引导观众视线到主体
- 创造画面深度
- 增加动态感

**TVC应用**:
```
使用手机拍摄时，利用道路线条引导到远处的产品
建筑走廊的透视线条引向品牌logo
手臂伸展的方向指向产品功能
```

## 高级构图技巧

### 5. 框架构图 (Frame within Frame)

**手法**:
- 窗户、门框作为外层框架
- 前景物体框住主体
- 几何形状创造画中画

**效果**:
- 突出主体
- 增加层次感
- 创造偷窥感/亲密感

**TVC示例**:
```
Shot 1: 从窗外拍摄室内使用场景（窗框构图）
Shot 2: 透过圆形门洞看到产品展示
Shot 3: 书架间隙中展示阅读灯产品
```

---

### 6. 负空间运用 (Negative Space)

**概念**:
主体周围的空白区域，"无"即是"有"。

**作用**:
- 突出主体
- 营造简约高级感
- 给视觉以呼吸
- 暗示品牌定位

**TVC应用**:
- Apple风格产品展示
- 奢侈品广告
- 极简主义品牌

**计算公式**:
```
主体占画面30%，负空间占70%
或 主体占画面40%，负空间占60%
```

---

### 7. 景深控制 (Depth of Field)

**浅景深 (Shallow DOF)**:
- f/1.4 - f/2.8
- 背景虚化，突出主体
- 人像、产品特写

**深景深 (Deep DOF)**:
- f/8 - f/16
- 前后景都清晰
- 风景、建筑、群体

**TVC叙事应用**:
```
开场: 深景深展示环境（ establishing shot）
中段: 浅景深聚焦产品（产品特写）
结尾: 中等景深展示使用场景（上下文）
```

## 镜头叙事语言

### 8. 镜头高度与心理

| 镜头高度 | 心理效应 | 适用场景 |
|----------|----------|----------|
| **鸟瞰/上帝视角** | 渺小、命运、全貌 | 宏大叙事、群体场景 |
| **高角度** | 弱势、无助、审视 | 儿童视角、被观察 |
| **平视** | 平等、真实、亲近 | 对话、日常场景 |
| **低角度** | 强大、威严、仰视 | 英雄产品、高端品牌 |
| **蚂蚁视角** | 压迫、宏大、危险 | 建筑、巨人视角 |

**TVC应用示例**:
```
低角度拍摄智能手机 = 产品如摩天大楼般宏伟
鸟瞰拍摄餐桌 = 展示产品在日常生活中的位置
平视拍摄用户表情 = 建立情感连接
```

| 比例 | 情感/用途 | 适用平台 |

## 镜头运动叙事

### 10. 推轨镜头 (Dolly/Track)

**推近 (Dolly In)**:
- 强调主体重要性
- 进入人物内心
- 细节放大

**拉远 (Dolly Out)**:
- 揭示环境上下文
- 抽离情感
- 宏大感

**TVC应用**:
```
开场拉远: 从脸部特写拉远到全身，展示产品使用场景
结尾推进: 推向产品logo，强调品牌
```

---

### 11. 手持镜头 (Handheld)

**效果**:
- 真实感、纪实感
- 紧张感、不稳定感
- 亲近感、现场感

**适用**:
- 纪录片风格广告
- 极限运动产品
- 真实用户场景

**注意事项**:
- 轻微抖动即可，避免眩晕
- 配合呼吸感，不要机械

## 光影构图

### 12. 伦勃朗光 (Rembrandt Lighting)

**特征**:
- 45度角主光
- 阴影侧脸上出现三角形光斑
- 戏剧感、立体感

**TVC应用**:
- 高端访谈
- 产品质感展示
- 戏剧性产品发布

---

### 13. 蝴蝶光/派拉蒙光 (Butterfly Lighting)

**特征**:
- 正上方主光
- 鼻下蝴蝶状阴影
- 美化人像、时尚感

**TVC应用**:
- 美妆产品
- 时尚品牌
- 美容仪器

---

### 14. 剪影 (Silhouette)

**效果**:
- 神秘感
- 形态强调
- 情绪表达

**TVC应用**:
```
日出剪影展示跑步者轮廓（运动产品）
逆光剪影展示产品轮廓（神秘感）
```

## 色彩构图

### 15. 色块构图

**手法**:
- 大面积纯色背景
- 产品作为色彩焦点
- 色彩对比或和谐

**TVC应用**:
- Apple风格纯色背景
- 时尚品牌色彩营销
- 产品颜色分类展示

## 实战应用检查清单

### 每个镜头自检

- [ ] **主体位置**: 是否使用三分法或黄金分割？
- [ ] **视线引导**: 是否有明确的视觉焦点？
- [ ] **深度层次**: 前景、中景、背景是否分明？
- [ ] **留白呼吸**: 负空间是否合适？
- [ ] **色彩平衡**: 色调是否统一且有对比？
- [ ] **光线质量**: 光影是否塑造立体感？
- [ ] **镜头高度**: 视角是否传达正确情绪？
- [ ] **运动目的**: 镜头运动是否有叙事理由？

## 经典TVC构图案例分析

### 案例1: Apple "Shot on iPhone"

**构图特点**:
- 简洁背景（大量负空间）
- 主体居中或三分点
- 自然光、真实场景
- 浅景深突出主体

**可学习点**:
- 少即是多
- 真实感胜过华丽
- 产品永远是主角

---

### 案例2: Nike "Find Your Greatness"

**构图特点**:
- 低角度拍摄（英雄视角）
- 动态模糊表现速度
- 特写与全景交替
- 强烈色彩对比

**可学习点**:
- 视角决定态度
- 运动感和力量的表现
- 情感高于产品展示

---

### 案例3: Chanel No.5

**构图特点**:
- 高对比度黑白
- 经典好莱坞布光
- 女性主体黄金分割
- 香水瓶对称构图

**可学习点**:
- 经典永不过时
- 奢华感的营造
- 品牌调性的一致性
