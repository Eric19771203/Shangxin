# 15-Second Modular Timing Framework

## Overview

TVC storyboards are built on 15-second modules that can be combined for 15s/30s/45s/60s durations. Each module has its own narrative arc while connecting to the overall story.

---

## Pacing Options

| Shot | Time | Duration |
|------|------|----------|
| 1 | 0-4s | 4s |
| 2 | 4-8s | 4s |
| 3 | 8-12s | 4s |
| 4 | 12-15s | 3s |

**Best For**: Premium, emotional, complex

## Module Connections

### Seamless Connect

End of Module A visual вҶ’ Start of Module B
- Match on action
- Match on color
- Match on subject

### Beat Break

Clear pause at module boundary
- 1-2 beat rest
- Music transition
- Visual pause

## Practical Examples

### 30s Standard (6 shots/15s)

```
MODULE A (0-15s):
Shot 1: 0-2s    - Product close-up (Hook)
```
