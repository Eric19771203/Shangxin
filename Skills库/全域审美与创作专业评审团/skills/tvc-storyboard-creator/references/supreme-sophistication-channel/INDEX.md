### 必须遵守的5大黄金法则

- [ ] **无卖点硬广**: 全程不讲产品参数、功能、优势
- [ ] **无台词洗脑**: 不使用旁白、解说、卖点口号
- [ ] **单一核心情绪**: 全片只传递1个核心情绪

- [ ] **"看不懂"即高级**: "看不懂"的门槛是高级感的来源

### 技术检查项

**拍摄前**:
- [ ] 场景符合品牌类型定位
- [ ] 道具选择有质感（无塑料、无杂乱）
- [ ] 光线测试完成（黄金时段时间确认）

**拍摄中**:
- [ ] 每个镜头≥3秒（15秒版）或≥5秒（30秒版）
- [ ] 无快剪、无卡点
- [ ] 主体占画面≤1/3
- [ ] 全程固定镜头或极慢推轨
- [ ] 无中景（无人物半身/全身）

**后期**:
- [ ] 无旁白、无台词、无slogan硬广
- [ ] 音乐音量≤-18dB
- [ ] 品牌Logo只在最后1-4秒出现
- [ ] 全片色彩≤4种
- [ ] 无产品功能卖点文字
