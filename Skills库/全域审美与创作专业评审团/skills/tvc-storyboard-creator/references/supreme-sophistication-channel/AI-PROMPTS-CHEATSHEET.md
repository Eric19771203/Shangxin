## 使用技巧

### 组合提示词公式
```
基础高级感词 + 场景类别词 + 时长专用词 + 品牌风格词 + 特殊效果词
```

**示例**:
```
[supreme sophistication aesthetic, minimalist composition, cinematic lighting] 
+ [natural poetic atmosphere, morning golden hour light] 
+ [30-second emotional progression visualization] 
+ [chanel elegance, timeless femininity] 
+ [extreme macro photography, shallow depth of field]
= 完整的香奈儿30秒TVC提示词
```

### 权重调整
- 想要更强调光影：在提示词中重复"lighting"相关词2-3次
- 想要更强调质感：在提示词中重复"texture"、"tactile"相关词
- 想要更强调情绪：在提示词中重复"mood"、"atmosphere"相关词

### 负面提示词（避免出现的元素）
```
no fast cuts, no text overlay, no slogan, no hard sell, no product specifications, no busy background, no harsh lighting, no oversaturation, no trendy filters, no social media stickers
```
