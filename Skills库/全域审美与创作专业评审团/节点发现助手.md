# 节点发现助手 (Node Discovery Assistant)

## 🎯 功能说明

节点发现助手是一个智能工具，帮助你快速找到合适的功能节点来完成任务。

---

## 📋 节点分类索引 (18个节点)

### 内容评审类 (2个)
- **NODE-001**: 内容评审节点 - 全品类内容专业评审、打分、优化建议
- **NODE-004**: 创意分析器 - 分析创意可行性、市场潜力和风险

### 创意生成类 (3个)
- **NODE-002**: 创意生成器 - 生成AI短剧创意，爽点选择、元素融合
- **NODE-003**: 故事类型转换大师 - 故事在不同类型间转换
- **NODE-015**: 角色创建专家 - 创建立体角色，含背景、性格、成长弧线

### 剧本创作类 (5个)
- **NODE-005**: 剧本创作大师 - 专业剧本创作，完整故事结构和角色塑造
- **NODE-006**: 爽文短剧编剧 - 爽文短剧创作，快速冲突与反转
- **NODE-007**: 网文改编短剧编剧 - 网文小说改编为短剧剧本
- **NODE-016**: 玄幻类漫剧编剧 - 玄幻题材漫剧，修炼体系、世界观构建
- **NODE-017**: 末世重生类漫剧编剧 - 末世重生题材漫剧，生存策略

### 小说创作类 (3个)
- **NODE-012**: 小说生成器 - 生成完整小说内容
- **NODE-013**: 短篇小说家 - 创作短篇小说，紧凑叙事
- **NODE-014**: 网文专业作家 - 创作网络小说，爽点设置

### 分镜创作类 (4个)
- **NODE-008**: 分镜头大师 - 剧本转换为详细分镜头脚本
- **NODE-009**: TVC广告分镜创作器 - TVC广告分镜，多区域适配、多视觉风格
- **NODE-010**: 分镜画面转AI绘画指令 - 分镜描述转AI绘画提示词
- **NODE-011**: 分镜画面转AI视频指令 - 分镜描述转AI视频生成提示词

### 改编类 (2个)
- **NODE-007**: 网文改编短剧编剧 - 网文改编短剧
- **NODE-018**: 网文改编漫剧专家 - 网文小说改编为漫剧形式
