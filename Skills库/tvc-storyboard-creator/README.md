# TVC Storyboard Creator v2.1.0

## 🎬 项目概述

**TVC Storyboard Creator** 是一个AI驱动的专业电视广告分镜生成系统，通过交互式工作流引导用户完成创意方向、视觉风格、节奏设计等多维度定制，最终生成包含镜头分解、时间轴、摄影指导和AI图像生成提示词的完整分镜文档。

**核心理念**：将产品输入转化为可直接用于制作的专业分镜文档。
