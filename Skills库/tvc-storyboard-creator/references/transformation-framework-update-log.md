## 新增功能概览

### 🆕 功能1：Transformation Framework Selection（转换框架选择）

**位置**：工作流程 Step 9.5（在创意想象力权重之后，脚本开发之前）

**功能描述**：
为用户提供三种创意转换框架，专门用于展现产品带来的状态/维度转变：

#### 框架A：Before-After-Bridge（前后对比）
- **结构**：Problem State → Transformation Moment → New Reality
- **适用场景**：
  - 护肤品：暗沉肌肤 → 光泽肌肤
  - 交通工具：拥堵通勤 → 顺畅旅程
  - 生活方式：高压日常 → 放松状态
- **时间分配**：Before (30-40%) → Bridge (20-30%) → After (30-40%)

#### 框架B：Dimensional Shift（维度变换）
- **结构**：Limited State → Breaking Boundaries → Expanded Reality
- **适用场景**：
  - "沉闷都市通勤" → "自由城际探索"
  - "受限空间" → "无限可能"
  - "平凡日常" → "非凡体验"
- **时间分配**：Dimension A (30%) → Transition (20%) → Dimension B (40%) → CTA (10%)
- **核心概念**：产品作为"维度之门"，让用户从一种现实跃迁到另一种现实

#### 框架C：Hybrid Transformation（混合转换）
- **结构**：Before in Dim A → Bridge → After in Dim B
- **优势**：
  - 结合Before-After的情感冲击
  - 加上Dimensional Shift的可能性扩展
  - 最大化展示产品的转变力量

**用户交互点**：
1. 呈现三种框架选项及相关案例
2. 用户选择框架
3. 定义具体转换：
   - Dimension A状态（如："沉闷都市通勤"）
   - Dimension B状态（如："自由城际探索"）
   - 转换触发点（产品如何实现转变）

## 框架使用指南

### 何时使用Before-After-Bridge？

**最佳产品类型**：
- 功能性强的产品（护肤、清洁、工具）
- 有明显效果对比的产品
- 解决具体痛点的B2C产品

**最佳视觉风格**：
- Bold Color Blocking（强烈色彩对比）
- Prada珠宝风（选择性色彩保留）
- Mocha Mousse（温暖大地色系）

**叙事重点**：
- 突出"问题有多严重"
- 产品作为"解决方案"
- "之后有多好"的 payoff

---

### 何时使用Dimensional Shift？

**最佳产品类型**：
- 拓展可能性的产品（交通工具、户外装备、科技产品）
- 代表生活方式转变的产品
- 高端/aspirational品牌

**最佳视觉风格**：
- Makoto Shinkai（开阔的天空和风景）
- Wes Anderson（对称构图，两个维度的对比）
- Cyberpunk（数字→现实维度的转换）

**叙事重点**：
- 突出"当前维度的局限"
- 产品作为"维度门户"
- "新维度的自由"

---

### 何时使用Hybrid Transformation？

**最佳产品类型**：
- 既有功能改进又有情感价值的产品
- 高价/高参与度产品（汽车、房产、奢侈品）
- 需要建立强烈情感连接的品牌

**叙事重点**：
- 既解决现实问题（Before-After）
- 又提供梦想可能（Dimensional Shift）
- 创造"必须拥有"的紧迫感

## 框架对比表

| 维度 | Before-After-Bridge | Dimensional Shift | Hybrid Transformation |
|------|---------------------|-------------------|----------------------|
| **核心隐喻** | 修复/改进 | 跃迁/穿越 | 重生/进化 |
| **情感诉求** | 解决问题 | 追求自由 | 彻底转变 |
| **视觉对比** | 状态对比（灰→彩） | 空间对比（封闭→开放） | 双重对比 |
| **产品定位** | 解决方案 | 门户/钥匙 | 转变催化剂 |
| **时间分配** | 30-40-30 | 30-20-40-10 | 30-20-50 |
| **最佳时长** | 15-30秒 | 30-60秒 | 45-60秒 |
| **适用品类** | 护肤、清洁、工具 | 交通、户外、科技 | 汽车、房产、奢侈品 |

## 摄影与后期指南

### Before-After-Bridge拍摄要点

**Before部分**：
- 镜头：长焦（85mm+）压缩空间感
- 光线：人工冷光，去饱和LUT
- 色调：灰蓝、灰褐、低对比度
- 构图：封闭、居中、对称

**Bridge部分**：
- 镜头：标准镜头（50mm）
- 光线：光线开始变暖
- 特效：产品发光、色彩开始渗透
- 运镜：从固定转为缓慢移动

**After部分**：
- 镜头：广角（24-35mm）扩展空间
- 光线：自然光或暖光，饱和LUT
- 色调：暖金、天空蓝、草地绿
- 构图：开放、三分法、层次丰富

---

### Dimensional Shift拍摄要点

**Dimension A（都市）**：
- 地点：CBD、地铁、电梯
- 时间：阴天或清晨
- 稳定器：少用，多用固定镜头
- 音效：城市噪音、机械重复

**Transition（穿越）**：
- 特效：像素撕裂、光门、色彩爆发
- 音效："咻"声+音乐转调
- 时长：必须足够明显（至少2秒）

**Dimension B（自然）**：
- 地点：海岸、山野、古镇
- 时间：黄金时段（日出后1h/日落前1h）
- 稳定器：全程使用，流畅运镜
- 音效：自然声音、开阔音乐

## 使用建议

### 对于创意总监
- 使用Hybrid Transformation建立品牌级的情感campaign
- 用Dimensional Shift打造aspirational品牌形象
- Before-After-Bridge适合功能导向的performance marketing

### 对于制作团队
- Dimensional Shift需要更多预算（双地点拍摄+特效）
- Before-After-Bridge最容易执行（单地点+调色即可）
- Hybrid Transformation需要平衡两种风格，确保过渡自然

### 对于客户
- Hybrid Transformation ROI最高（情感+功能双重说服）
- Dimensional Shift适合品牌长期建设
- Before-After-Bridge适合短期销售转化
