# English Film Terminology

## Shot Types / Camera Angles

| Term | Abbreviation | Description |
|------|--------------|-------------|
| Close-up | CU | Tight shot of subject, usually face |
| Medium Close-up | MCU | From chest up |
| Medium Shot | MS | Waist up |
| Medium Wide | MW | Knees up |
| Wide Shot | WS | Full body or wider |
| Extreme Close-up | ECU | Very tight detail |
| Over the Shoulder | OTS | Behind one person to another |
| Point of View | POV | Camera as character's eyes |
| Two-shot | 2S | Two subjects in frame |
| Insert | INS | Detail shot of object |
| Cutaway | CA | Related cut to other action |
| Establishing | EST | Sets up location/scene |

---

## Camera Movement

| Term | Description |
|------|-------------|
| Pan | Horizontal camera rotation |
| Tilt | Vertical camera rotation |
| Dolly | Camera moves toward/away |
| Truck | Camera moves left/right |
| Pedestal | Camera moves up/down |
| Zoom | Lens magnification (not movement) |
| Tracking | Following subject movement |
| Crane | Elevated moving shot |
| Handheld | Camera operator holds |
| Steadicam | Gyro-stabilized camera |

---

## Composition

| Term | Description |
|------|-------------|
| Rule of thirds | Composition grid 3x3 |
| Headroom | Space above subject |
| Lead room | Space in front of subject |
| Framing | How subjects are positioned |
| Symmetry | Balanced composition |
| Depth | Foreground/background layers |
| Negative space | Empty areas |
| Background | Behind subject |
| Foreground | In front of subject |

---

## Lighting

| Term | Description |
|------|-------------|
| Key light | Main light source |
| Fill light | Reduces shadows |
| Rim light | Edge separation |
| Back light | Behind subject |
| Practical | Visible light sources |
| High key | Bright overall |
| Low key | Dark, shadows |
| Rembrandt | Single source triangle |

## Audio

| Term | Description |
|------|-------------|
| VO | Voice-over narration |
| Narration | Storyteller voice |
| Diegetic | Sound from scene |
| Non-diegetic | Added in post |
| SFX | Sound effects |
| BGM | Background music |
| Ambient | Environmental sound |
| Mixing | Combining audio tracks |
