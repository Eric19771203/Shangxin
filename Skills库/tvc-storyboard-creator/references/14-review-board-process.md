## 评审团组成

### 核心评审角色

#### 1. 创意总监 (Creative Director)
**职责**：
- 评估创意的原创性和突破性
- 确保创意符合品牌调性
- 把控整体创意方向

**评审重点**：
- 创意是否独特
- 是否符合品牌DNA
- 能否在竞品中脱颖而出

#### 2. 资深编剧 (Senior Screenwriter)
**职责**：
- 评估剧本结构和叙事节奏
- 检查对话的自然度和感染力
- 确保故事逻辑自洽

**评审重点**：
- 三幕结构是否完整
- 情感弧光是否清晰
- 对话是否精炼有力

#### 3. 导演 (Director)
**职责**：
- 评估剧本的可拍性
- 提出镜头语言建议
- 把控视觉风格统一性

**评审重点**：
- 画面描述是否清晰
- 镜头设计是否可行
- 视觉节奏是否得当

#### 4. 品牌代表 (Brand Representative)
**职责**：
- 确保品牌信息准确传达
- 检查品牌元素使用规范
- 评估市场接受度

**评审重点**：
- 品牌信息是否突出
- 产品展示是否自然
- 是否符合目标市场文化

#### 5. 消费者洞察专家 (Consumer Insights Specialist)
**职责**：
- 评估目标受众的共鸣度
- 预测市场反应
- 确保情感连接有效

**评审重点**：
- 目标受众能否共情
- 情感触动是否真实
- 行动号召是否有效

## 评审流程

### 完整评审流程图

```
剧本大纲 → 创意与剧本评审 → 修改 → 详细剧本 → 导演与编剧复检 → 修改 → 分镜剧本 → 最终确认
```

### 第一阶段：创意与剧本评审

#### 评审时机
- 剧本大纲完成后
- 详细剧本初稿完成后

#### 评审步骤
1. **材料准备**
   - 剧本大纲/详细剧本
   - 创意说明文档
   - 参考资料（视觉、音乐等）

2. **独立评审**（24小时）
   - 每位评审独立阅读
   - 填写评审表
   - 记录初步意见

3. **集体评审会**（90分钟）
   - 创意陈述（10分钟）
   - 评审意见发表（每人8分钟）
   - 自由讨论（30分钟）
   - 形成共识意见（10分钟）

4. **反馈整理**
   - 综合所有评审意见
   - 优先级排序
   - 形成可执行的修改建议

### 第二阶段：导演与编剧复检

#### 评审时机
- 详细剧本修改完成后
- 分镜剧本初稿完成后

#### 评审重点
- 剧本的可执行性
- 镜头语言的完整性
- 技术层面的可行性
- 与最终产出的衔接

## 创意与剧本评审

### 评审维度

#### 1. 创意维度 (25分)

| 评分项 | 评分标准 | 权重 |
|--------|----------|------|
| 原创性 | 是否有突破性创意，不雷同于竞品 | 10 |
| 相关性 | 创意与品牌/产品是否强关联 | 8 |
| 扩展性 | 创意是否适合多平台、多版本延展 | 7 |

#### 2. 叙事维度 (25分)

| 评分项 | 评分标准 | 权重 |
|--------|----------|------|
| 结构完整性 | 三幕结构是否清晰，起承转合是否到位 | 10 |
| 节奏把控 | 15秒内节奏是否张弛有度 | 8 |
| 情感张力 | 故事是否有感染力，能否触动观众 | 7 |

#### 3. 角色维度 (15分)

| 评分项 | 评分标准 | 权重 |
|--------|----------|------|
| 认同感 | 主角是否能让目标受众产生认同 | 8 |
| 立体感 | 角色是否有层次，不是扁平的 | 7 |

#### 4. 品牌维度 (20分)

| 评分项 | 评分标准 | 权重 |
|--------|----------|------|
| 信息准确 | 品牌核心信息是否准确传达 | 8 |
| 融入自然 | 品牌/产品是否自然融入故事 | 7 |
| 调性一致 | 风格是否与品牌调性相符 | 5 |

#### 5. 执行维度 (15分)

| 评分项 | 评分标准 | 权重 |
|--------|----------|------|
| 可拍性 | 剧本是否具备可执行性 | 8 |
| 成本可控 | 制作成本是否在合理范围内 | 7 |

### 评审问题清单

#### 创意评审问题
- [ ] 这个创意的核心洞察是什么？
- [ ] 在3秒内能否吸引目标受众的注意力？
- [ ] 与竞品相比，独特性在哪里？
- [ ] 创意是否有足够的情感深度？
- [ ] 创意是否有记忆点？

#### 剧本评审问题
- [ ] 故事的起点、发展、高潮、结局是否清晰？
- [ ] 每一秒的时间利用是否合理？
- [ ] 对话是否精炼，每一句都有存在的必要？
- [ ] 视觉描述是否足够具体，可以直接转化为画面？
- [ ] 情感弧光是否完整，观众的情感变化是否可预期？

## 导演与编剧复检

### 导演复检维度

#### 1. 镜头语言
- [ ] 场景设计是否有视觉冲击力？
- [ ] 景别变化是否丰富且合理？
- [ ] 镜头运动是否服务于叙事？
- [ ] 构图是否专业且有美感？
- [ ] 光线设计是否营造了合适的氛围？

#### 2. 视觉节奏
- [ ] 镜头时长分配是否合理？
- [ ] 剪辑节奏是否与叙事节奏匹配？
- [ ] 转场设计是否流畅有创意？
- [ ] 快慢节奏对比是否得当？

#### 3. 技术可行性
- [ ] 所需特殊效果是否可实现？
- [ ] 拍摄场地是否可获得？
- [ ] 演员表演要求是否合理？
- [ ] 道具制作是否可行？
- [ ] 后期制作周期是否足够？

### 编剧复检维度

#### 1. 叙事完整性
- [ ] 修改后的剧本是否保持了原创意的核心？
- [ ] 故事逻辑是否自洽，没有漏洞？
- [ ] 情感线是否连贯，没有断裂？
- [ ] 结局是否令人满意且有回味？

#### 2. 细节打磨
- [ ] 每一个动作描述是否必要？
- [ ] 每一句对话是否有力量？
- [ ] 每一个场景是否推动剧情发展？
- [ ] 是否有冗余的内容可以删除？

#### 3. 与分镜的衔接
- [ ] 剧本是否为分镜创作提供了足够的空间？
- [ ] 关键画面是否在剧本中有明确描述？
- [ ] 时间标注是否精确到0.1秒？
- [ ] 声音设计是否与画面同步？

## 评审提示词库

### System Role - 创意总监评审

```
你是一位拥有20年经验的4A广告公司创意总监，曾带领团队获得戛纳金狮奖、克里奥大奖、亚太广告节金奖等50+国际奖项。你以眼光毒辣、要求严苛著称，但总能提出极具建设性的修改意见。

你的评审风格：
1. 直击要害 - 不绕弯子，直接指出问题
2. 既破又立 - 指出问题的同时给出解决方案
3. 鼓励创新 - 欣赏突破性的想法，即使不完美
4. 品牌导向 - 始终以品牌利益为最高原则

请从创意总监的角度评审以下内容。
```

### System Role - 资深编剧评审

```
你是一位好莱坞级别的资深编剧，曾为多部获奖电影和广告撰写剧本。你精通叙事技巧，善于在极短篇幅内讲述动人的故事。

你的评审重点：
1. 结构严谨 - 三幕结构是否完整
2. 节奏精准 - 每一秒都经过精心设计
3. 情感真挚 - 故事能触动人心
4. 对话精炼 - 每一句都有力量

请从专业编剧的角度评审以下内容。
```

### System Role - 导演评审

```
你是一位荣获戛纳金棕榈提名的电影导演，同时也是国际知名的广告导演。你善于用镜头语言讲故事，对视觉美学有极高的追求。

你的评审角度：
1. 画面思维 - 每一个文字都能转化为画面
2. 镜头设计 - 景别、运动、构图的巧妙运用
3. 光线氛围 - 用光影营造情绪
4. 可执行性 - 创意是否能真正落地

请从导演的角度评审以下内容。
```

### 创意与剧本评审提示词

```
请作为评审团对以下剧本进行评审：

[在此处粘贴剧本内容]

## 评审要求

请从以下五个维度进行评审，每个维度包含具体问题：

### 1. 创意维度
- 这个创意的核心洞察是什么？
- 创意的独特性在哪里？与竞品相比有何不同？
- 创意是否有记忆点？3年后观众还能记得吗？
- 创意是否适合多平台延展？

### 2. 叙事维度
- 三幕结构是否清晰？起承转合是否到位？
- 15秒的时间分配是否合理？每一秒都有价值吗？
- 情感弧光是否完整？观众的情感变化是怎样的？
- 节奏是否张弛有度？快慢结合是否得当？

### 3. 角色维度
- 主角是否能让目标受众产生认同？
- 角色的动机是什么？是否合理？
- 角色是否有成长或转变？

### 4. 品牌维度
- 品牌核心信息是否准确传达？
- 产品展示是否自然？不是生硬的植入？
- 风格调性是否与品牌相符？

### 5. 执行维度
- 剧本的可拍性如何？有没有难以实现的场景？
- 制作成本是否在合理范围内？
- 拍摄周期是否可控？
```
