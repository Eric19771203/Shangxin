## 剧本格式规范

### 标准剧本格式要素

#### 1. 场景标题 (Scene Heading)
```
INT. 闺房 - 黄昏
EXT. 江南庭院 - 清晨
```

- INT. = 内景 (Interior)
- EXT. = 外景 (Exterior)
- INT./EXT. = 内外景结合
- 时间：黎明/清晨/上午/中午/下午/黄昏/夜晚/深夜

#### 2. 动作描述 (Action)
- 现在时态
- 简洁有力
- 只写能看到和听到的
- 每段不超过4行

#### 3. 角色名 (Character)
- 全大写
- 居中
- 首次出现加粗

#### 4. 对话 (Dialogue)
- 角色名下方
- 缩进
- 每段不超过3行

#### 5. 旁白/画外音 (Voiceover)
```
胭脂 (V.O.)
色彩，是灵魂的语言。
```

#### 6. 配乐/音效 (Sound)
```
[悠扬的古琴声]
[水滴落下的清脆声]
```

## 三幕剧结构

### 第一幕：建置 (Setup) - 0-5秒
**目标**：建立情境，引入主角，呈现冲突

#### 关键元素
1. **钩子 (Hook)** - 第1秒抓住注意力
2. **主角介绍** - 建立人物认同
3. **情境设定** - 时间、地点、氛围
4. **激励事件** - 打破平衡

### 第二幕：对抗 (Confrontation) - 5-12秒
**目标**：升级冲突，展示转变，推向高潮

#### 关键元素
1. ** rising action** - 逐步升级
2. ** midpoint** - 重大转折
3. ** obstacles** - 克服障碍
4. ** climax build** - 推向高潮

### 第三幕：解决 (Resolution) - 12-15秒
**目标**：解决冲突，展示成果，呼吁行动

#### 关键元素
1. **高潮 (Climax)** - 决定性时刻
2. **结局 (Denouement)** - 展示结果
3. **品牌展示** - 产品亮相
4. **行动号召 (CTA)** - 引导购买

## TVC剧本创作框架

### 15秒TVC标准时间分配

| 时间点 | 时长 | 内容 | 目的 |
|--------|------|------|------|
| 0:00-0:02 | 2s | 视觉钩子 | 抓住注意力 |
| 0:02-0:06 | 4s | 问题/情境 | 建立共鸣 |
| 0:06-0:11 | 5s | 解决方案/转变 | 展示价值 |
| 0:11-0:14 | 3s | 品牌/产品 | 强化记忆 |
| 0:14-0:15 | 1s | 行动号召 | 引导行动 |

### TVC剧本创作 Checklist

#### 视觉元素
- [ ] 每一秒都有视觉变化
- [ ] 色彩设计有品牌识别度
- [ ] 构图专业（三分法、黄金比例）
- [ ] 光线营造氛围
- [ ] 有标志性画面

#### 叙事元素
- [ ] 有清晰的情感弧光
- [ ] 主角有目标/欲望
- [ ] 有转折或惊喜
- [ ] 结局令人满意
- [ ] 品牌融入自然

#### 技术元素
- [ ] 时间精确到0.1秒
- [ ] 画面/声音同步
- [ ] 适合目标平台
- [ ] 有可识别的品牌元素
- [ ] 符合目标市场文化

## 专业剧本写作提示词

### System Role - 专业编剧

```
你是一位荣获戛纳金狮奖、克里奥奖、亚太广告节金奖的顶级TVC编剧，拥有15年专业经验。你精通东西方叙事技巧，擅长用15秒讲述感人至深的故事。

你的创作原则：
1. 视觉优先 - 每一句话都能转化为画面
2. 情感驱动 - 触动人心最柔软的部分
3. 品牌自然 - 产品是故事的有机组成部分
4. 文化尊重 - 深入理解目标市场的文化语境
5. 节奏精准 - 15秒的每一刻都经过精心设计

请根据以下要求创作专业剧本。
```
