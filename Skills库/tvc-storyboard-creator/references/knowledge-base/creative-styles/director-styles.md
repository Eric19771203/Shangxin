## 亚洲导演

### 王家卫 (Wong Kar-wai)

**视觉特征**:
- **色彩**: 高饱和度、霓虹色调、红绿对比
- **光影**: 慢快门拖影、朦胧感、雨夜反光
- **构图**: 倾斜构图、前景遮挡、多层空间
- **情绪**: 暧昧、疏离、时间流逝感

**代表作品**:
- 《花样年华》: 旗袍、走廊、克制欲望
- 《重庆森林》: 罐头、凤梨、都市孤独
- 《2046》: 科幻怀旧、记忆碎片

**TVC应用**:
- 高端香水、酒类广告
- 奢侈品牌情感诉求
- 城市生活品牌

**AI提示词**:

## 使用指南

### 如何选择导演风格

1. **产品属性匹配**
   - 高端奢侈 → 王家卫、诺兰
   - 家庭温情 → 是枝裕和
   - 潮流年轻 → 昆汀
   - 科技未来 → 诺兰、科幻风格

2. **目标受众匹配**
   - 文艺青年 → 王家卫、独立电影风格
   - 大众市场 → 斯皮尔伯格、温暖风格
   - 潮流先锋 → 昆汀、实验风格

3. **品牌调性匹配**
   - 东方美学 → 张艺谋、王家卫
   - 西方现代 → 诺兰、芬奇
   - 日式简约 → 是枝裕和

### 风格混搭建议

- 王家卫 + 赛博朋克 = 霓虹都市情感
- 张艺谋 + Wes Anderson = 东方对称美学
- 诺兰 + 产品展示 = 科技产品史诗感
