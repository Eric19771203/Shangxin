# TVC Storyboard Creator - 知识库索引

## 快速导航

本知识库系统提供专业级的创意、摄影、广告知识，支持TVC分镜创作。

---

## 知识库结构

### 📁 creative-styles/ 创意风格库
**用途**: 寻找视觉灵感、风格参考、创意方向

| 文件 | 内容 | 使用场景 |
|------|------|----------|
| `director-styles.md` | 王家卫、张艺谋、诺兰等20+导演风格 | 需要特定视觉风格时 |
| `fashion-trends-2025-2026.md` | 最新时尚广告趋势：大胆色彩分割、科技浪漫主义、Prada珠宝风、雕刻光效等 | 需要前沿视觉风格时 |

**关键词搜索**:
- 东方美学 → 王家卫、张艺谋
- 科技感 → 诺兰、赛博朋克
- 极简 → 北欧、日式
- 潮流前沿 → **Bold Color Blocking、Fashion Romantasy、Prada Jewelry Style**
- 产品冲击 → **Sculptural Lighting、Liquid Metal、Digital Twin**
- 2025年度色 → **Mocha Mousse (摩卡慕斯)**

### 📁 tvc-industry/ 广告行业库
**用途**: 广告策略、消费者心理、平台规范

| 文件 | 内容 | 使用场景 |

### 🤖 learning-system/ 自学习系统 ⭐NEW
**用途**: 持续优化推荐算法，越用越智能

| 文件 | 内容 | 使用场景 |
|------|------|----------|
| `README.md` | 完整的学习系统设计文档、算法原理、实施路线图 | 了解AI学习机制 |
| `learning_engine.py` | Python实现代码，包含数据收集、分析、优化 | 技术实现参考 |
| `learning-data/` | 训练数据和性能统计 | 查看学习成果 |

- 🔄 A/B测试（验证新算法效果）
- 📦 版本管理（模型版本控制、回滚机制）

**监控指标**:
- 推荐采纳率目标: >70%
- Top3采纳率目标: >90%
- 满意度目标: >4.2/5
- 决策时间目标: <30秒

## 使用场景速查

### 场景1: 需要创意灵感
1. 打开 `creative-styles/director-styles.md`
2. 根据产品属性选择导演风格
3. 提取AI提示词关键词
4. 应用到分镜创作

### 场景2: 分镜构图不专业
1. 打开 `cinematography/composition-techniques.md`
2. 查看构图法则（三分法、黄金分割等）
3. 应用到当前分镜镜头
4. 检查镜头语言是否传达正确情绪

### 场景3: 不了解目标市场
1. 打开 `tvc-industry/advertising-psychology.md`
2. 查看消费者心理学章节
3. 了解AIDA模型应用
4. 查看平台特定策略

### 场景4: AI生成图像质量差
1. 打开 `product-photography/lighting-techniques.md`
2. 找到对应产品的布光方案
3. 提取专业摄影术语
4. 优化AI提示词
