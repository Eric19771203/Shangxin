# Narrative Review Checklist

## Overview

This checklist ensures storyboard quality, narrative coherence, and production readiness.

---

## 1. Narrative Coherence

### Story Flow
- [ ] Clear beginning, middle, end
- [ ] Logical progression between shots
- [ ] Smooth transitions between scenes
- [ ] No jump cuts that confuse
- [ ] Time flow is clear

### Shot-to-Shot Logic
- [ ] Each shot leads naturally to next
- [ ] Match on action where appropriate
- [ ] No jarring jumps in subject/position
- [ ] Camera movements justified

### Module Connections (30s+)
- [ ] Each module has complete mini-story
- [ ] Modules connect seamlessly
- [ ] No redundant information
- [ ] Cumulative narrative builds

---

## 2. Timing Accuracy

### Total Duration
- [ ] Total time matches selected (15/30/45/60s)
- [ ] Shot durations are realistic
- [ ] No over/under timing issues

### Individual Shots
- [ ] Minimum shot length respected (1.5s minimum)
- [ ] Complex shots have adequate time
- [ ] Transitions calculated (0.25-0.5s)

### Pacing
- [ ] Pacing matches selected style (cinematic/standard/dynamic)
- [ ] Rhythm appropriate for product
- [ ] No dragging or rushing

---

## 3. Information Hierarchy

### Key Message
- [ ] Main message clear and prominent
- [ ] USP featured adequately
- [ ] Not lost in details

### Product Visibility
- [ ] Product clearly shown
- [ ] Logo appears at end
- [ ] Key features demonstrated

### Call to Action
- [ ] Clear CTA present
- [ ] CTA in final shots
- [ ] Action specified

---

## 4. Emotional Arc

### Progression
- [ ] Emotional tone appropriate
- [ ] Builds to satisfying conclusion
- [ ] Not flat throughout

### Engagement
- [ ] Hook captures attention
- [ ] Maintains interest
- [ ] Ends memorably

### Audience Match
- [ ] Tone matches target audience
- [ ] Regional appropriateness
- [ ] Cultural sensitivity

---

## 5. Visual Quality

### Composition
- [ ] Clear subjects
- [ ] Appropriate framing
- [ ] Good use of space

### Visual Interest
- [ ] Variety in shot types
- [ ] Not repetitive
- [ ] Dynamic where needed

### Style Adherence
- [ ] Consistent visual style
- [ ] Style matches brand
- [ ] Color palette appropriate

---

## 6. Technical Considerations

### Camera Directions
- [ ] Feasible camera movements
- [ ] Equipment appropriate
- [ ] Lighting achievable

### Production Reality
- [ ] Locations realistic
- [ ] Budget appropriate
- [ ] Schedule feasible

### AI Prompts
- [ ] Prompts generateable
- [ ] Style consistent
- [ ] Technical specifications clear

---

## 7. Cultural Sensitivity

### Regional Appropriateness
- [ ] Regional values respected
- [ ] No offensive content
- [ ] Appropriate symbols

### Legal/Compliance
- [ ] Claims substantiated
- [ ] No false advertising
- [ ] Rights cleared
