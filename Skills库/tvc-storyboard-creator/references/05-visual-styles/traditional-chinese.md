# Traditional Chinese Ink Painting Style Guide

## Overview

Traditional Chinese ink painting (水墨画) features brush-stroke aesthetics, negative space, and monochromatic compositions with poetic atmosphere.

---

## Visual Characteristics

### Color Palette

**Primary**:
- Black ink (#000000)
- Various greys
- White (negative space)

**Accent**:
- Seal red (#ff0000)
- Subtle blue
- Muted earth tones

### Key Elements

- Brush stroke texture
- Mountain/water imagery
- Calligraphic elements
- Negative space
- Poetic composition
