## 改编原则

### 核心原则

#### 1. 视觉优先 (Visual First)
- 每一个场景都要转化为具体的画面
- 文字描述要能让导演和摄影师立即想象出镜头
- 用画面讲故事，而不是用文字解释

#### 2. 节奏精确 (Precise Rhythm)
- 每个镜头的时长精确到0.1秒
- 镜头数量与时长匹配（15秒≈8-12个镜头）
- 节奏变化服务于情感曲线

#### 3. 技术可行 (Technically Feasible)
- 考虑实际拍摄的可能性
- 平衡创意与可执行性
- 为后期制作留足空间

#### 4. 品牌强化 (Brand Reinforcement)
- 关键画面突出品牌元素
- 产品展示自然且有记忆点
- 色彩、构图呼应品牌调性

## 改编流程

### 六步改编法

```
第1步：剧本拆解 → 第2步：镜头规划 → 第3步：分镜创作 → 第4步：技术标注 → 第5步：Gemini适配 → 第6步：Seance适配
```

#### 第1步：剧本拆解

**任务**：将详细剧本拆解为可独立拍摄的场景单元

**输出**：
- 场景清单
- 关键情节点
- 情感转折点
- 品牌展示点

**方法**：
1. 通读剧本3遍以上
2. 标记每个情节点
3. 识别情感高潮
4. 确定品牌亮相时刻
5. 列出所有场景转换

#### 第2步：镜头规划

**任务**：为每个场景设计镜头序列

**输出**：
- 镜头清单（Shot List）
- 景别设计
- 镜头运动方案
- 时长分配

**考虑因素**：
- 叙事节奏
- 情感表达
- 视觉丰富度
- 拍摄可行性

#### 第3步：分镜创作

**任务**：为每个镜头撰写详细的分镜描述

**输出**：
- 完整分镜剧本
- 画面描述
- 声音设计
- 技术说明

**标准**：
- 画面描述具体可拍
- 技术参数明确
- 时长精确到0.1秒
- 与前后镜头衔接流畅

#### 第4步：技术标注

**任务**：添加完整的技术标注

**输出**：
- 摄影参数
- 灯光设计
- 美术要求
- 后期指示

**标注内容**：
- 镜头编号
- 时长
- 景别
- 镜头运动
- 构图
- 光线
- 色彩
- 转场

#### 第5步：Gemini 3适配

**任务**：将分镜剧本转化为Gemini 3多宫格分镜提示词

**输出**：
- Gemini 3系统提示词
- 多宫格布局（3×3或2×4）
- 每个镜头的详细描述
- 风格统一说明

**关键要素**：
- 专业的场景描述（中/英双语）
- 四大核心参数（景别、运镜、构图、光线）
- 统一的视觉风格
- 清晰的镜头编号

#### 第6步：Seance 2.0适配

**任务**：将分镜剧本转化为Seance 2.0视频生成提示词

**输出**：
- Seance 2.0系统提示词
- 完整的时间轴序列
- 每个镜头的视频提示词
- 转场和连续性要求

**关键要素**：
- 精确的时间码（00:00.0-00:02.0格式）
- 主体/动作描述
- 七大核心元素（时间码、主体、镜头、环境、光线、风格、转场）
- 物理现实主义和连续性保证

## 镜头设计策略

### 景别选择策略

#### 景别功能表

| 景别 | 功能 | 适用场景 | 情感效果 |
|------|------|----------|----------|
| **大远景 (ELS)** | 建立环境、展示规模 | 开场、场景转换 | 宏大、孤独、自由 |
| **远景 (LS)** | 展示人物与环境关系 | 人物入场、场景展示 | 客观、观察 |
| **全景 (FS)** | 展示人物全身动作 | 完整动作展示 | 完整、连贯 |
| **中景 (MS)** | 展示上半身动作与表情 | 对话、反应 | 交流、互动 |
| **中近景 (MCU)** | 重点展示面部表情 | 重要情绪时刻 | 亲密、关注 |
| **近景 (CU)** | 突出细节与情绪 | 关键反应、产品特写 | 紧张、亲密 |
| **特写 (ECU)** | 极致细节展示 | 产品细节、眼神 | 冲击、强调 |

#### 15秒TVC景别推荐序列

**经典模式**：
```
大远景/远景 → 中景 → 近景 → 特写 → 中景 → 近景 → 产品特写 → 品牌Logo
```

**情感递进模式**：
```
远景（建立）→ 中景（引入）→ 近景（深入）→ 特写（高潮）→ 中景（收束）→ 产品特写（品牌）
```

### 镜头运动策略

#### 运动类型与效果

| 运动类型 | 功能 | 情感效果 | 适用场景 |
|----------|------|----------|----------|
| **推 (Push-in)** | 强调、聚焦 | 紧张、关注 | 重要时刻、发现 |
| **拉 (Pull-out)** | 揭示、展开 | 释然、开阔 | 场景展示、结局 |
| **移 (Track/Dolly)** | 跟随、探索 | 流畅、探索 | 人物跟随、环境探索 |
| **摇 (Pan)** | 扫视、展示 | 观察、展示 | 环境展示、多元素 |
| **升降 (Crane/Jib)** | 宏大、升华 | 宏大、升华 | 开场、高潮、结局 |
| **旋转 (Rotate)** | 混乱、眩晕 | 激动、混乱 | 转折、梦幻 |
| **手持 (Handheld)** | 真实、紧迫 | 真实、紧张 | 纪录片风格、行动 |
| **固定 (Static)** | 稳定、客观 | 冷静、观察 | 正式、重要时刻 |

#### 运动节奏设计

**快节奏**（兴奋、活力）：
- 快速推拉
- 手持晃动
- 快速剪辑
- 短镜头（0.5-1.0秒）

**慢节奏**（优雅、抒情）：
- 缓慢推移
- 流畅轨道
- 长镜头（2.0-3.0秒）
- 舒缓的升降

### 构图策略

#### 16种专业构图技法（详见 composition-techniques.md）

1. **三分法 (Rule of Thirds)** - 经典平衡
2. **黄金比例 (Golden Ratio)** - 自然美感
3. **对称构图 (Symmetry)** - 正式庄重
4. **引导线 (Leading Lines)** - 引导视线
5. **框架构图 (Framing)** - 聚焦主体
6. **负空间 (Negative Space)** - 简约高级
7. **前景运用 (Foreground)** - 增加层次
8. **背景简化 (Simplify Background)** - 突出主体
9. **视角变化 (Perspective)** - 创造新意
10. **重复图案 (Repetition)** - 节奏感
11. **色彩对比 (Color Contrast)** - 视觉冲击
12. **明暗对比 (Chiaroscuro)** - 戏剧效果
13. **深度创造 (Depth)** - 三维感
14. **动态对称 (Dynamic Symmetry)** - 能量感
15. **打破规则 (Rule Breaking)** - 创意突破
16. **平衡控制 (Balance)** - 视觉稳定

#### TVC常用构图组合

**开场画面**：
- 三分法 + 引导线 + 负空间
- 营造氛围，引导视线

**产品展示**：
- 对称构图 + 居中 + 黄金比例
- 突出产品，正式大气

**人物互动**：
- 三分法 + 前景 + 深度创造
- 自然真实，层次丰富

**情感高潮**：
- 打破规则 + 明暗对比 + 色彩对比
- 创造冲击，强化记忆

### 光线设计策略

#### 光线类型与效果

| 光线类型 | 功能 | 情感效果 | 适用场景 |
|----------|------|----------|----------|
| **自然光 (Natural)** | 真实、自然 | 亲切、真实 | 日景、外景 |
| **硬光 (Hard)** | 锐利、清晰 | 硬朗、现代 | 产品特写、质感 |
| **柔光 (Soft)** | 柔和、唯美 | 优雅、浪漫 | 人物、情感 |
| **侧光 (Side)** | 塑造、立体 | 戏剧、神秘 | 人物塑造 |
| **逆光 (Backlight)** | 轮廓、梦幻 | 梦幻、圣洁 | 高光时刻 |
| **顶光 (Top)** | 戏剧、神秘 | 神秘、紧张 | 特殊效果 |
| **底光 (Bottom)** | 怪异、恐怖 | 不安、怪异 | 创意效果 |

#### TVC灯光设计模式

**优雅模式**（奢侈品、美妆）：
- 柔光主光 + 柔光辅光 + 轮廓光
- 光比 2:1，柔和唯美

**现代模式**（科技、时尚）：
- 硬光主光 + 弱辅光 + 强轮廓光
- 光比 4:1，锐利现代

**戏剧模式**（高端、艺术）：
- 侧光主光 + 弱补光 + 效果光
- 光比 8:1，戏剧张力

**自然模式**（日常、亲民）：
- 模拟自然光 + 柔光补光
- 光比 1.5:1，亲切真实

### 案例：胭脂口红 - 详细剧本到分镜改编

## 关键技巧总结

### 快速改编 Checklist

- [ ] 通读剧本3遍以上
- [ ] 识别关键情节点（3-5个）
- [ ] 规划镜头数量（8-12个）
- [ ] 设计景别序列（远→中→近→特）
- [ ] 选择镜头运动（服务于叙事）
- [ ] 运用专业构图（每种镜头1-2种技法）
- [ ] 设计光线方案（营造氛围）
- [ ] 精确时长分配（总计15秒）
- [ ] 声音同步设计（画面+声音）
- [ ] 技术标注完整（摄影/美术/后期）

### 常见问题解决方案

| 问题 | 解决方案 |
|------|----------|
| 镜头太多太碎 | 合并相似镜头，每个镜头要有明确目的 |
| 节奏太慢 | 增加短镜头（0.5-1.0秒），加快运动 |
| 节奏太快 | 增加长镜头（2.0-3.0秒），用固定或慢运动 |
| 画面单调 | 变化景别、运动、构图，丰富视觉语言 |
| 品牌生硬 | 提前规划品牌展示点，自然融入叙事 |
| 技术不可行 | 平衡创意与执行，提供备选方案 |
