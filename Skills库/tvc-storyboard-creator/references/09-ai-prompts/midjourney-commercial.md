# AI Image Generation Prompts - Midjourney

## Overview

This guide provides optimized Midjourney prompts for generating TVC storyboard images.

---

## Prompt Structure

### Base Formula

```
[Subject] + [Scene/Setting] + [Lighting] + [Camera/Composition] + [Style Modifiers] + --ar [Aspect Ratio] --v [Version]
```

### Standard Components

```
[Product/Subject], 
[Cinematic lighting], 
[Professional commercial photography], 
[High resolution], 
[Detailed],
[--ar 16:9 --v 6]
```

## Parameters Reference

### Aspect Ratios

| Ratio | Use |
|-------|-----|
| --ar 16:9 | Standard TVC |
| --ar 9:16 | Vertical social |
| --ar 4:5 | Instagram |
| --ar 1:1 | Square |

### Version Flags

| Version | Description |
|---------|-------------|
| --v 6 | Latest version |
| --v 5.2 | Previous stable |
| --style raw | Less interpretation |

### Quality Flags

| Flag | Effect |
|------|--------|
| --q 2 | Higher quality |
| --s 750 | Higher stylize |
| --iw 0.8 | Image weight |

---

## Best Practices

1. **Be Specific**: Include exact product details
2. **Lighting First**: Describe light source first
3. **Style Consistent**: Match all shots to selected style
4. **Regional**: Add regional keywords when needed
5. **Test**: Generate variations before final

---

## Common Prompt Issues

### Avoid

- Too many style keywords
- Contradictory styles
- Ambiguous subject descriptions
- Missing lighting information

### Include

- Product clearly described
- Intended mood
- Key style elements
- Technical specs (lighting, camera)
