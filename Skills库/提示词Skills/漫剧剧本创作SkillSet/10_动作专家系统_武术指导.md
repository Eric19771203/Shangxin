# 10_动作专家系统_武术指导 (Action Expert System: Martial Arts & Choreography)

## 核心功能
漫剧武术指导+动作专家系统，从根源解决AI动作戏的空泛同质化、无逻辑、与人设/世界观脱节、无法落地分镜、无情绪张力等核心痛点。

## 模块一：角色专属动作指纹系统
为每个核心角色生成**唯一、固定、全程不更改的动作指纹库**，从根源上解决「所有角色打起来一个样」的AI通病。
- **指纹维度**：核心格斗体系、标志性动作/杀招、攻防逻辑习惯、动作细节习惯、能力联动规则、动作禁忌红线、情绪动作映射。
- **全自动生成提示词**：
```markdown
你是拥有20年影视动作指导经验的漫剧专属动作专家，基于我提供的角色人设、世界观、能力体系、阶级出身，为该角色生成【专属动作指纹库】。
要求：
1. 严格按照「核心格斗体系、标志性动作/杀招、攻防逻辑习惯、动作细节习惯、能力联动规则、动作禁忌红线、情绪动作映射」7个维度生成
2. 所有规则必须100%贴合角色的性格、出身、能力、世界观设定
3. 必须明确该角色的「动作禁忌红线」，全程严格规避
4. 动作设计必须适配漫剧分镜、竖屏画面呈现
5. 最终输出的动作指纹库，可直接用于后续所有剧本动作戏的创作与校验
```

## 模块二：漫剧动作戏专属节奏控制体系
针对竖屏漫剧的手机观看场景，拆解出精准到秒的动作戏节奏公式。
- **60秒竖屏漫剧节奏**：
  - 0-3s 起势危机（特写冲击镜头）
  - 3-12s 攻防试探（中景跟镜，主角先落入劣势）
  - 12-22s 冲突升级（近景快切，主角陷入绝境）
  - 22-28s 反转杀招（慢镜头+特写，核心爽点爆发）
  - 28-30s 收尾钩子（全景固定，留下悬念）

## 模块三：专业级动作描述标准化模板
- **通用标准化模板**：
```
【镜号】【景别】【运镜】【时长】【画面/动作时序】【能力联动】【身体细节/微表情】【音效】【情绪/潜台词】【对应台词】
```
- **全自动优化提示词**：
```markdown
你是拥有20年影视武术指导经验的漫剧动作专家，基于我提供的角色动作指纹库、世界观能力体系、单集剧情、分镜节奏，对剧本中的动作戏做专业级优化。
要求：
1. 严格按照「1个镜头对应1个动作节点」的规则，使用标准化模板
2. 所有动作必须严格匹配对应角色的动作指纹库
3. 动作设计必须严格贴合世界观的能力体系、物理规则
4. 严格遵循竖屏漫剧动作戏节奏铁律，动作时长、镜头切换完全匹配节奏公式
5. 彻底删除空泛的文学化描述
6. 最终输出「优化前后对比表」+「标准化动作分镜剧本」
```

## 模块四：动作-镜头-台词-情绪四维绑定系统
- **动作与情绪绑定**：动作是情绪的具象化，愤怒时动作加快/破绽多，隐忍时动作收敛/防守反击。
- **动作与台词绑定**：动作间隙说台词，单句不超7个字，台词语气匹配动作力度。
- **动作与镜头绑定**：竖屏优先用特写/近景，快攻用快切，杀招用慢镜头。
