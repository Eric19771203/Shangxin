# 11_武器专家系统_道具设计 (Weapon Expert System: Prop Design)

## 核心功能
漫剧武器专家系统，从根源解决AI武器道具的乱用错用、参数失真、与人设/世界观脱节、无剧情作用、无法落地画面等核心痛点。

## 模块一：角色专属武器身份卡系统
为每个核心角色生成**唯一、固定、全程不更改的武器身份卡**，所有武器使用必须严格遵循卡片规则。
- **身份卡维度**：基础档案（名称/型号/参数）、改装/专属细节、核心使用规则、世界观能力联动、携带与操作习惯、武器禁忌红线、情绪映射规则、成长与叙事线。
- **全自动生成提示词**：
```markdown
你是拥有30年军工/冷兵器研究经验的漫剧专属武器专家，基于我提供的角色人设、世界观、能力体系、格斗体系，为该角色生成【专属武器身份卡】。
要求：
1. 严格按照「基础档案、改装/专属细节、核心使用规则、世界观能力联动、携带与操作习惯、武器禁忌红线、情绪映射规则、成长与叙事线」8个维度生成
2. 所有规则必须100%贴合角色的出身、阶级、性格、格斗体系、世界观设定
3. 必须明确武器的精准参数、使用逻辑、禁忌红线，全程严格规避专业常识错误
4. 必须设计固定的视觉辨识度细节
5. 必须设计武器的剧情叙事线
```

## 模块二：武器使用刚性规则与专业校验体系
- **热兵器规则**：明确型号/口径/弹容量，操作逻辑（拔枪→开保险→瞄准），场景适配（室内手枪/远距离狙击）。
- **冷兵器规则**：明确全长/重量，攻击逻辑（刀劈/刺击），材质损耗。
- **全自动专业校验提示词**：
```markdown
你是漫剧武器专业校验师，拥有军工、冷兵器、影视道具设计30年专业经验，基于我提供的角色武器身份卡、世界观设定、剧本内容，执行武器专业度全维度校验。
要求：
1. 专业常识校验：检查武器参数、操作逻辑、伤害逻辑
2. 人设匹配校验：检查武器是否匹配身份卡
3. 世界观适配校验：检查是否符合世界观技术水平
4. 动作逻辑校验：检查武器与格斗动作是否匹配
5. 一致性校验：检查视觉特征是否统一
```

## 模块三：专业武器描述标准化模板
- **通用标准化模板**：
```
【镜号】【景别】【运镜】【时长】【武器操作时序】【武器细节特写】【能力联动】【身体/微表情细节】【音效】【情绪/潜台词】【对应台词】
```
- **分题材专业模板**：涵盖现代热兵器、武侠冷兵器、赛博朋克特殊武器。

## 模块四：武器-动作-情绪-台词四维绑定系统
- **武器-动作联动**：动作设计贴合武器特性，操作间隙匹配镜头节奏。
- **武器-情绪联动**：情绪变化影响武器操作精度，武器特写放大情绪张力。
- **武器-台词联动**：操作间隙说短台词，台词带出武器背后的故事。
