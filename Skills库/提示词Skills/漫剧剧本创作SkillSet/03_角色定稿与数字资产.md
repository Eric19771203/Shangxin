# 03_角色定稿与数字资产 (Character Finalization & Digital Assets Skill)

## 核心功能
基于世界观，自动生成全剧角色定稿设定，同时构建可复用的角色数字资产库，彻底解决漫剧全流程角色一致性崩图的行业痛点。

## 输入内容（总控中枢自动传入）
- 世界观设定文档、IP核心冲突、全剧角色清单

## 输出成果
- 全角色官方定稿设定集（和你之前定制的角色定稿模板完全匹配）、角色多角度定妆图、角色数字资产包（特征向量、seed值、reference图、提示词模板）、角色与世界观绑定校验报告

## 配套工具
- 大模型：Claude 3.5 Sonnet、GPT-4o
- 视觉工具：Midjourney、Stable Diffusion、即梦AI
- 资产管理工具：ComfyUI角色库、ELSER.AI

## 自动化执行流程
1.  自动基于世界观核心矛盾，生成全角色定稿设定集（核心锚点、基础档案、外貌细节、性格弧光、能力设定、服化道）
2.  自动生成角色定妆提示词，批量生成角色正面、45度、侧面三视图，锁定角色核心五官、造型、配色、seed值
3.  自动构建角色数字资产包，将角色特征编码为固定向量，确保后续所有画面生成100%匹配角色设定，一致性达92%以上
4.  自动提交总控中枢校验，通过后传入剧本创作Skill

## 核心落地提示词（直接复制）
```markdown
你是专业漫剧角色设定师，基于上方的世界观与IP定位，生成全剧角色官方定稿设定集。
要求：
1. 每个角色必须包含：核心锚点（绑定世界观核心矛盾）、基础档案、3D建模级外貌细节、性格与人物弧光、能力与规则设定、服化道定稿、核心辨识度特征
2. 每个角色必须生成【固定特征字典】，明确3个以上全剧不更改的视觉辨识度特征，确保全流程一致性
3. 每个角色必须生成标准化AI生图提示词模板，可直接用于画面生成
4. 人设必须与世界观强绑定，角色的核心困境、动机、成长弧光必须由世界观规则塑造，杜绝人设与剧情脱节
5. 输出完成后，自动执行人设与世界观匹配度校验，输出校验报告
```

## 行业验证核心优化点
- 每个角色必须绑定世界观核心矛盾，角色的核心困境、动机、成长弧光，必须由世界观规则直接塑造
- 固定3个以上核心视觉辨识度特征（疤痕、配饰、发型、瞳色），全剧不更改
- 资产包内置「角色一致性控制方案」：SD/ComfyUI用Reference Only+IP-Adapter，权重0.7-0.8；Midjourney用--cref + --cw 100
