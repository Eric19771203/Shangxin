# 01_爆款定位与IP孵化 (Hit Positioning & IP Incubation Skill)

## 核心功能
用大数据验证赛道可行性，锁定爆款方向，避免自嗨创作，为全流程定调，是漫剧能否起量的核心前提。

## 输入内容（总控中枢自动传入）
- 漫剧题材、目标平台（抖音/快手/B站/小红书）、目标受众、核心风格

## 输出成果
- 赛道爆款数据分析报告、对标爆款拆解、核心高概念、3秒抓眼钩子、IP核心卖点、全剧核心爽点/虐点框架

## 配套工具
- 数据工具：新抖、新快、蝉妈妈、灰豚数据
- 大模型：Claude 3.5 Sonnet、GPT-4o

## 自动化执行流程
1.  自动爬取目标平台近30天同题材TOP100爆款漫剧，拆解核心数据：完播率、点赞率、评论关键词、核心爽点、节奏结构、人设标签
2.  自动筛选出「高热度、低竞争」的细分赛道，输出赛道可行性报告
3.  自动拆解对标爆款的核心公式，生成适配你的IP高概念、核心钩子、爽点框架
4.  自动提交总控中枢校验，通过后传入世界观搭建Skill

## 核心落地提示词（直接复制）
```markdown
你是专业漫剧IP定位专家，拥有抖音/快手TOP100漫剧爆款操盘经验，基于【XX题材】赛道，生成可落地的漫剧IP定位方案。
要求：
1. 拆解该赛道近30天爆款的核心共性：人设标签、爽点结构、3秒开篇钩子、单集节奏公式、核心冲突、结尾留坑技巧
2. 筛选高热度低竞争的细分方向，输出IP核心高概念（一句话讲清故事核心看点与差异化）
3. 明确全剧核心爽点框架、单集节奏标准、目标受众、核心卖点
4. 所有内容严格适配竖屏短视频漫剧的传播逻辑，拒绝自嗨，可直接用于后续世界观搭建
```
