# 09_双专家系统_台词与情感优化 (Dual Expert System: Dialogue & Emotion)

## 核心功能
漫剧双专家系统包含「台词专家」和「情感专家」，从根源上解决AI剧本的同质化、模板化、情绪空洞、不说人话、人设脱节等核心AI感问题。

## 漫剧台词专家系统
核心逻辑：**先给角色固定「语言指纹」，再给台词定「漫剧专属规则」，最后做「镜头绑定优化」，从根源上避免AI同质化**。

### 模块1：角色专属语言指纹系统
在角色定稿环节，提前为每个核心角色生成**唯一、固定、全程不更改的语言指纹库**。
- **指纹维度**：基础用词习惯、句式结构、专属口癖/口头禅、语速与停顿规则、情绪表达句式、禁忌表达红线。
- **全自动生成提示词**：
```markdown
你是漫剧专属台词专家，拥有10年头部国漫台词创作经验，基于我提供的角色人设、世界观、阶级站位，为该角色生成【专属语言指纹库】。
要求：
1. 严格按照「基础用词习惯、句式结构、专属口癖/口头禅、语速与停顿规则、情绪表达句式、禁忌表达红线」6个维度生成
2. 所有规则必须100%贴合角色的出身、阶级、性格、核心执念，完全匹配世界观设定
3. 必须明确该角色的「绝对禁用词/禁用句式」，全程严格规避
4. 最终输出的语言指纹库，可直接用于后续所有剧本台词的创作与校验
```

### 模块2：漫剧专属台词节奏优化模块
竖屏漫剧的台词核心规则：**单句台词不超15字，1个镜头对应1句台词，台词永远服务于画面**。
- **全自动优化提示词**：
```markdown
你是漫剧专属台词专家，基于我提供的角色语言指纹库、漫剧单集时长、目标平台，对剧本台词做竖屏漫剧专属优化，彻底消除AI感。
要求：
1. 严格遵循4条刚性铁律：单句台词不超15字，1个镜头对应1句台词，总字数严格匹配时长，1句台词只传递1个信息
2. 彻底删除所有书面语、长难句、大段独白、信息灌输式台词，改成符合角色语言指纹库的口语化表达
3. 能靠画面、动作传递的信息/情绪，直接删掉对应的台词，只保留必须用台词说的内容
4. 优化后的台词，必须严格匹配每个角色的语言指纹库
5. 最终输出「优化前后对比表」+「优化后的最终台词剧本」
```

### 模块3：潜台词与动作绑定系统
让每一句台词都对应动作、对应潜台词，彻底消除AI的直白空洞感。
- **全自动优化提示词**：
```markdown
你是漫剧专属台词专家，基于我提供的剧本、角色语言指纹库、分镜镜头，对台词做潜台词与动作绑定优化。
要求：
1. 为每一句核心台词，匹配对应的镜头动作、微表情
2. 为每一句冲突台词、情绪台词，标注核心潜台词，确保台词有言外之意
3. 彻底删除所有直白的情绪呐喊、信息复述式台词
4. 优化后的台词，必须严格匹配角色的语言指纹库
5. 最终输出「优化后的台词剧本」，标注每一句的对应动作、潜台词
```

## 漫剧情感专家系统
核心逻辑：**人类的情绪永远是复杂的、分层的**，从「角色情绪分层→观众共情锚点→情绪节奏匹配」全链路优化。

### 模块1：角色分层情绪表达系统
为每个角色、每个剧情节点，生成分层情绪表达规则（主情绪+次情绪+隐藏情绪）。
- **全自动优化提示词**：
```markdown
你是漫剧专属情感专家，基于我提供的剧本、角色人设、剧情节点，为核心台词做分层情绪优化。
要求：
1. 为每一句核心情绪台词，拆解「主情绪+次情绪+隐藏情绪」的3层结构
2. 基于分层情绪，优化台词的表达，彻底删除直白的情绪呐喊、模板化的情绪台词
3. 严格遵循角色的情绪表达边界，确保情绪表达完全贴合人设，不OOC
4. 为每一句优化后的台词，标注对应的分层情绪、微表情、动作细节
5. 最终输出「优化前后对比表」+「分层情绪优化后的最终台词剧本」
```

### 模块2：观众共情锚点设计系统
在每一集设计1-2个精准的观众共情锚点，让台词戳中观众的普遍情绪。
- **全自动设计提示词**：
```markdown
你是漫剧专属情感专家，基于我提供的单集剧本、核心剧情、目标受众，为本集设计1-2个精准的观众共情锚点，优化对应的台词。
要求：
1. 共情锚点必须贴合本集的核心剧情、主角的人物弧光
2. 锚点台词必须戳中目标受众的普遍情绪，不能只服务于主角的个人情绪
3. 锚点台词必须简短、有力、符合角色的语言指纹库
4. 最终输出「本集共情锚点设计方案」+「优化后的锚点台词」+「对应的镜头、动作、情绪设计」
```
