# 02_世界观全自动搭建 (Worldview Construction Skill)

## 核心功能
基于定位方向，自动生成逻辑闭环、可落地、适配漫剧叙事的完整世界观，完全匹配你之前定制的世界观搭建标准。

## 输入内容（总控中枢自动传入）
- IP定位报告、核心高概念、题材赛道、核心冲突框架

## 输出成果
- 完整世界观设定文档（核心规则、地理空间、权力体系、文化细节、历史背景、冰山式呈现方案）、世界观逻辑漏洞自查报告

## 配套工具
- 大模型：Claude 3.5 Sonnet、GPT-4o、豆包4.0
- 结构化工具：飞书文档、Notion数据库

## 自动化执行流程
1.  自动基于IP定位，调用你之前定制的世界观提示词模板，生成完整世界观初稿
2.  自动执行「逻辑自洽校验」，排查规则漏洞、前后矛盾、与定位不符的内容，自动优化
3.  自动拆解「观众可见10%内容」和「底层90%逻辑」，生成漫剧叙事的冰山式呈现方案
4.  自动提交总控中枢校验，通过后传入角色定稿Skill

## 核心落地提示词（直接复制）
```markdown
你是专业漫剧世界观构建大师，基于上方的IP定位方案，生成一套逻辑自洽、可直接支撑漫剧连载的完整世界观。
要求：
1. 必须包含：核心定调、世界基础规则、地理与空间设定、社会与权力体系、文化与日常细节、历史与背景叙事、角色与世界观绑定规则、冰山式呈现方案
2. 核心规则全程统一，无前后矛盾、无规则漏洞，所有设定必须服务于核心冲突与剧情推进，拒绝无效自嗨设定
3. 所有内容可直接视觉化，能转化为漫剧场景、服化道、剧情冲突，无抽象描述
4. 输出完成后，自动执行逻辑漏洞自查，输出校验报告，有问题自动优化
```

## 深度拓展·世界观建立专业体系

### 一、世界观底层逻辑重构：从「设定堆砌」到「冲突驱动的闭环体系」
核心工具：**世界观冲突金字塔模型**。
| 冲突层级 | 核心定义 | 刚性要求 |
|----------|----------|----------|
| 第5层：底层规则冲突 | 世界核心规则本身的天然矛盾 | 必须是不可调和的二元对立 |
| 第4层：文明/阵营冲突 | 基于底层规则形成的势力对立 | 必须有明确的利益诉求、权力边界 |
| 第3层：社会/阶级冲突 | 基于阵营格局形成的社会规则、阶级壁垒 | 必须可视觉化、可通过日常剧情体现 |
| 第2层：群体/人际冲突 | 基于社会规则形成的角色间对立 | 必须绑定上层冲突，不能是脱离世界观的私人恩怨 |
| 第1层：个体/内心冲突 | 角色自身与世界观规则的对抗 | 必须完全贴合底层规则，是主角所有选择的终极动机 |

### 二、世界观全维度对抗验证体系（可全自动落地）
| 验证维度 | 验证目标 | 全自动提示词模板 |
|----------|----------|------------------|
| 逻辑自洽对抗验证 | 排查规则漏洞、前后矛盾 | ```你是专业世界观逻辑校验师...用反证法，假设每一条核心规则被打破...``` |
| 冲突驱动力对抗验证 | 确保冲突有层级、有根源 | ```你是专业漫剧冲突校验师...检查5层冲突是否完全打通...``` |
| 角色绑定对抗验证 | 确保角色与世界观强绑定 | ```你是专业角色设定校验师...检查每个核心角色的核心困境、动机...``` |
| 视觉落地对抗验证 | 确保所有设定可转化为漫剧画面 | ```你是专业漫剧分镜师...逐条检查所有设定，是否能通过画面呈现...``` |
| 连载容错对抗验证 | 确保世界观有拓展空间 | ```你是专业漫剧连载策划师...检查是否预留了支线剧情、新角色...``` |
