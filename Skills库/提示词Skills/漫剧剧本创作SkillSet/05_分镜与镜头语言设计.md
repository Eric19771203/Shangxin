# 05_分镜与镜头语言设计 (Storyboard & Camera Language Skill)

## 核心功能
把剧本自动转化为「AI可精准执行的分镜指令」，解决行业内「剧本和画面脱节」的痛点，用影视级镜头语言强化漫剧叙事张力。

## 输入内容（总控中枢自动传入）
- 单集分镜剧本、角色数字资产、世界观场景设定

## 输出成果
- 影视级分镜脚本（含镜头坐标、运镜参数、景别、构图、光影、画面提示）、分镜故事板、AI生图精准提示词

## 配套工具
- 大模型：Claude 3.5 Sonnet、GPT-4o
- 分镜工具：StoryboardGen、商汤Seko、剪映分镜模板

## 自动化执行流程
1.  自动将剧本拆解为单个镜头，为每个镜头匹配最优景别、运镜、构图，采用头部团队验证的「三维空间坐标系」标注镜头位置，避免越轴跳切
2.  自动为每个镜头生成精准的AI生图提示词，包含角色特征、场景、动作、光影、构图、画质标准，和你之前的角色定稿、线稿渲染提示词完全匹配
3.  自动生成分镜故事板，标注镜头切换方式、时长、音效节点，可直接用于后续画面生成
4.  自动提交总控中枢校验，通过后传入视觉生成Skill

## 核心落地提示词（直接复制）
```markdown
你是影视级漫剧分镜师，基于上方的单集剧本，生成可直接被AI绘画工具识别的标准化分镜脚本。
要求：
1. 每个镜头必须明确标注：镜号、时长、景别、运镜方式、拍摄角度、构图、画面内容、光影要求、角色动作表情、AI生图提示词
2. 严格遵循影视级镜头语言规则，避免越轴、跳切，用镜头强化叙事张力，动作戏、情绪戏匹配对应的镜头节奏
3. 每个镜头的AI生图提示词必须包含固定的角色特征、场景细节、画质标准，和之前的角色设定完全匹配
4. 输出完成后，自动执行镜头逻辑校验、AI可执行性校验，输出校验报告
```

```
镜头(15°俯角，轨道左移@2m/s，全景)，角色A(左勾拳动作，轨迹抛物线@30°，愤怒表情)，角色B(后退躲避，惊恐表情)，场景(废弃仓库，昏暗顶光，灰尘粒子漂浮)，电影级光影，景深拉满，8K超清，超拟真3D质感，角色五官100%匹配参考图，无崩坏无畸变
```
