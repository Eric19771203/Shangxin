# 04_剧本全自动创作 (Script Creation Skill)

## 核心功能
基于世界观和角色，自动生成符合漫剧爆款节奏、可直接落地分镜的完整剧本，从全剧大纲→分集大纲→单集分镜剧本全链路自动化。

## 输入内容（总控中枢自动传入）
- 世界观设定、角色数字资产、IP定位爽点框架、全剧集数、单集时长

## 输出成果
- 全剧主线大纲、分集大纲、单集标准化分镜剧本（含场景、人物、动作、台词、景别提示、情绪标注）、剧情逻辑校验报告

## 配套工具
- 大模型：Claude 3.5 Sonnet、GPT-4o、讯飞星火（双模型互补）
- 剧本工具：阅文漫剧助手、番茄小说IP改编工具

## 自动化执行流程
1.  自动生成全剧主线大纲，采用爆款公式：世界观+主角缺陷→激励事件→核心目标→三次行动/受阻→转折→最终觉醒→高潮→结局
2.  自动拆分解锁分集大纲，每集严格遵循「3秒抓眼危机→15秒交代背景→30秒核心冲突→1分钟反转推进→结尾强钩子」的竖屏漫剧节奏
3.  自动生成单集标准化分镜剧本，格式适配AI工具识别，标注清楚镜号、景别、动作、台词、情绪、画面提示，无抽象描述
4.  自动执行剧情逻辑校验，排查人设OOC、剧情拖沓、冲突不足、钩子不强的问题，自动优化
5.  自动提交总控中枢校验，通过后传入分镜设计Skill

## 核心落地提示词（直接复制）
```markdown
你是头部漫剧金牌编剧，拥有100+爆款漫剧创作经验，基于上方的世界观、角色设定、IP定位，生成漫剧标准化剧本。
要求：
1. 先输出全剧主线大纲，严格遵循爆款公式：主角缺陷+世界观困境→激励事件→核心目标→三次行动/受阻→转折→最终觉醒→高潮→结局
2. 再拆分解锁分集大纲，每集严格遵循「3秒抓眼危机→15秒背景交代→30秒核心冲突→1分钟反转推进→结尾强钩子」的竖屏漫剧节奏
3. 最后输出单集标准化分镜剧本，格式为：【镜号】【景别】【时长】【场景】【画面/动作】【台词/OS】【情绪】【音效提示】，无抽象描述，可直接用于分镜设计
4. 全程人设统一，无OOC，剧情逻辑自洽，每集冲突明确，钩子足够强，适配竖屏用户的观看习惯
5. 输出完成后，自动执行节奏校验、人设校验、逻辑校验，输出校验报告，不合格自动优化
```

## 深度拓展·剧本创作工业化体系

### 一、剧本核心结构升级：双主线并行
- **外部冲突主线（剧情推进线）**：由世界观核心矛盾驱动，主角为了实现核心目标，与反派/世界规则对抗的主线。
- **内部成长主线（人物弧光线）**：由主角的内心冲突驱动，主角在对抗外部困境的过程中，克服自身缺陷、完成自我和解。
- **双主线同步推进公式**：
  > **本集外部冲突事件 → 主角做出关键选择 → 外部冲突得到推进/反转 → 主角的内心困境得到暴露/解决 → 主角完成一次小的成长 → 为下一集冲突埋下伏笔**

### 二、全层级剧本节奏控制体系（精准到秒）
- **宏观节奏（全剧黄金螺旋）**：起势钩子（1-5集）、上升升级（6-20集）、高潮爆发（21-30集）、收尾留坑（31-32集）。
- **中观节奏（分平台分集）**：
  - 抖音/快手60秒：0-3s抓眼钩子 → 3-12s背景交代 → 12-30s冲突升级 → 30-50s反转爽点 → 50-60s结尾钩子。
- **微观节奏（镜头级情绪波浪）**：竖屏漫剧每15秒必须有一个情绪峰值，动作戏镜头0.5-2秒，情绪戏镜头3-5秒。

### 三、剧本全链路对抗验证体系（可全自动落地）
| 验证维度 | 全自动提示词模板 |
|----------|------------------|
| 人设OOC对抗验证 | ```你是专业剧本人设校验师...验证角色的选择是否符合他的性格...``` |
| 冲突合理性对抗验证 | ```你是专业剧本冲突校验师...验证本集冲突是否比上一集更强...``` |
| 节奏与情绪对抗验证 | ```你是专业漫剧节奏校验师...拆解本集每10秒的内容，绘制观众情绪曲线...``` |
| 逻辑闭环对抗验证 | ```你是专业剧本逻辑校验师...用反证法排查剧情中的逻辑漏洞...``` |
| 前置受众反馈对抗验证 | ```你是目标受众模拟测试器...模拟【18-25岁，喜欢XX题材漫剧的抖音用户】...``` |
