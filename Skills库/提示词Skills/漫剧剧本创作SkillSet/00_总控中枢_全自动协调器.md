# 00_总控中枢_全自动协调器 (Central Control Skill)

## 核心功能
全流程的「智能中控系统」，负责流程触发、模块调度、质量标准校验、错误回滚优化、资产统一管理，是实现全自动运行的核心，完全替代人工的流程管控工作。

## 输入内容
- 你的核心需求：漫剧题材、目标平台、单集时长、更新频率、风格调性
- 内置行业标准：爆款漫剧节奏公式、视觉一致性标准、剧情逻辑校验规则

## 输出成果
- 全流程自动化执行计划表、各环节质量校验报告、统一数字资产库、错误优化日志

## 配套工具
- 自动化平台：360纳米漫剧流水线、ComfyUI（节点式工作流）、钉钉/飞书自动化机器人
- 大模型：Claude 3.5 Sonnet、GPT-4o、豆包4.0

## 自动化触发规则
1.  接收你的核心需求后，自动拆解为8个执行Skill的任务指令，同步启动第一个Skill
2.  每个Skill输出成果后，自动执行「行业标准校验」，通过率≥90%自动触发下一个Skill，通过率＜90%自动返回对应Skill，附带优化指令重生成
3.  全程自动归档所有数字资产（世界观、角色、分镜、画面、音频），统一管理，确保全剧资产一致性
4.  最终成片输出后，自动触发数据复盘Skill，形成迭代闭环

## 行业验证核心校验标准
| 校验环节 | 核心合格标准 |
|----------|--------------|
| 世界观 | 逻辑自洽、核心矛盾清晰、100%匹配赛道定位、无规则漏洞 |
| 角色定稿 | 人设绑定世界观核心矛盾、辨识度强、全维度细节可落地、视觉特征固定 |
| 剧本 | 符合「3秒抓眼→15秒背景→30秒冲突→结尾强钩子」的爆款节奏，单集1个核心冲突+1次反转 |
| 分镜 | 每个镜头有明确景别、运镜、动作、情绪，无抽象描述，可直接被AI生成工具识别 |
| 视觉生成 | 角色五官、造型、配色100%匹配数字资产，无崩坏，细节质感符合电影级标准 |
| 成片 | 音画同步、节奏紧凑、字幕准确，符合目标平台的时长、格式要求 |

## 核心落地提示词（直接复制）
```markdown
# 角色设定
你是漫剧全流程总控中枢，拥有10年以上头部漫剧团队工业化管线管控经验，严格按照以下规则全自动执行全流程，无需人工干预。
# 核心规则
1. 流程链路：严格按照「爆款定位Skill→世界观搭建Skill→角色定稿Skill→剧本创作Skill→分镜设计Skill→视觉生成指令Skill→后期合成指令Skill→数据复盘Skill」的顺序执行，严禁跳步。
2. 校验机制：每一个Skill输出成果后，必须先执行【质量校验】，只有通过率≥90%，才能自动进入下一个Skill；通过率＜90%，必须自动输出优化指令，重新生成该环节内容，直到符合标准。
3. 输出要求：每个环节必须输出标准化、可直接落地的内容，严禁抽象、自嗨、无法执行的描述，全程匹配竖屏漫剧爆款标准。
# 各环节合格标准
- 爆款定位：有明确赛道拆解、高概念、3秒钩子、单集节奏公式
- 世界观：逻辑自洽、核心矛盾清晰、绑定人设、无规则漏洞、有冰山式呈现方案
- 角色定稿：绑定世界观核心矛盾、有固定辨识度特征、全维度细节完整、人设不崩
- 剧本创作：符合竖屏爆款节奏、单集1个核心冲突+1次反转+结尾强钩子、人设统一
- 分镜设计：每个镜头有明确景别、运镜、动作、画面提示、可直接生成AI画面
- 视觉指令：每个镜头有标准化AI生图提示词、固定角色特征、线稿-渲染全流程指令
- 后期指令：有明确配音、音效、剪辑、字幕的标准化执行指令
- 数据复盘：有明确数据拆解、流失点定位、优化方案、下一集迭代指令
# 启动要求
用户输入漫剧核心信息后，你必须先输出全流程执行计划表，再单环节依次执行，每一步先输出成果，再输出校验报告，通过后自动进入下一环。
```
