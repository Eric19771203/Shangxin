# 全域审美与创作专业评审 SKILL

## 1. 简介
本 SKILL 是一个基于国内顶级文艺奖项、国际电影节/广告节评审规范打造的**全品类创作专业评审工具**。它模拟由 **1位终审主席 + 16位分领域专家** 组成的评审团，为剧本、影视、短视频、广告等全品类创作提供 **艺术+心理+商业+合规** 四维深度的专业评审与量化评分。

## 2. 核心架构
本 SKILL 采用“角色扮演 + 量化评分 + 风险预警”的架构：

*   **评审团 (Expert Panel)**：
    *   **终审主席**：负责最终评分校准、争议裁定。
    *   **艺术创作赛道**：视觉、叙事、剧作、表演、声音、纪实、动漫专家。
    *   **时尚创意赛道**：时尚主编、创意总监。
    *   **产业新媒体赛道**：影评人、新媒体操盘手、制片人。
    *   **商业合规赛道**：投资人、法务专家。
    *   **受众心理赛道**：社会心理、两性心理专家。

*   **评分体系 (Scoring System)**：
    *   **基础分 (150分)**：15个核心维度，单项满分10分。
    *   **商业加权**：独立计算，系数 0.1-0.3。
    *   **附加/扣分**：原创奖励与违规惩罚。

## 3. 运行模式
本 SKILL 建议按照以下步骤使用：

1.  **提交作品**：使用 `/review` 指令，提供作品内容（文本/大纲/链接/描述）。
2.  **全维度评审**：AI 将扮演评审团，从16个专业维度进行独立点评。
3.  **获取报告**：生成包含综合评分、等级判定、风险提示、优化方案的完整报告。

## 4. 角色与职责

### 4.1 评审团主席 (Chairman)
*   **职责**：最终评分校准、争议裁定、核心价值定性。
*   **能力**：全维度平衡、顶层价值判断。

### 4.2 分领域专家 (Specialists)
*   **职责**：在各自专业领域内独立评审，出具单项评分与风险提示。
*   **能力**：视觉美学、剧作结构、商业评估、心理洞察、合规审核等。

## 5. 核心文件清单
*   `SKILL.md`: 核心配置文件，定义元数据、能力和接口。
*   `methodology.md`: 评审规则与评分体系。
*   `roles/`: 专家角色定义文件。
*   `templates/`: 评审报告模板。
*   `prompts/`: AI角色扮演指令。

## 6. 使用指南
1.  **初始化**：用户输入 `/review [作品内容]` 启动评审。
2.  **查看阵容**：输入 `/roles` 查看当前评审团专家。
3.  **查看标准**：输入 `/criteria` 查看评分维度。
