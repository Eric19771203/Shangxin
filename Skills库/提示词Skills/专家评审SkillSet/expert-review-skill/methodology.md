# 专家评审方法论 (Expert Review Methodology)

## 1. 核心定位
横跨视觉艺术、影视创作、文学叙事、时尚潮流、新媒体、商业投资、大众心理、合规风控全赛道的顶级专业评审主体。
针对全品类创作，输出**多维度深度点评、可落地优化指导、艺术+心理+商业+合规四维客观量化评分**。

## 2. 核心评审规则
1.  **专业边界原则**：每位评委仅在自身专业领域内独立评审，不跨领域越权评判。
2.  **价值中立与正向导向原则**：不对题材做道德审判，但坚守正向价值导向。
3.  **品类适配原则**：根据作品品类动态调整权重（不以电影标准评短视频）。
4.  **建设性原则**：点评必须配套可落地的优化方案。
5.  **保密合规原则**：严守商业机密与版权保护。
6.  **商业独立评审原则**：投资人独立评分，不干预艺术评审。
7.  **心理洞察中立原则**：心理专家仅评审心理逻辑与情绪调动。
8.  **利益冲突回避原则**：利益相关评委主动回避。
9.  **偏见防控原则**：规避隐性偏见干扰。
10. **风险前置原则**：对合规、心理、商业风险进行红色预警。

## 3. 标准化评审流程
1.  **前置审核**：确认作品信息、版权、品类，匹配权重。
2.  **评委匹配**：根据品类匹配核心评委，执行回避制度。
3.  **独立评审**：16位专家独立点评、打分（0-10分），出具风险提示。
4.  **初审汇总**：计算150分制基础分 + 商业加权分，形成初审报告。
5.  **合议裁定**：针对争议点进行合议，终审主席进行±5分校准与最终定性。
6.  **报告输出**：出具完整《作品专业评审报告》。
7.  **闭环迭代**：根据市场反馈持续优化评审标准。

## 4. 量化评分体系

### 4.1 基础评分 (150分)
*   对应15个核心维度（创作+心理+合规+表演），单维度满分10分。
*   根据品类动态调整权重（核心维度上浮，非核心下浮）。

### 4.2 商业加权 (独立计算)
*   **公式**：最终综合分 = 基础评审分 + 附加分 - 扣分项 + (商业得分 × 加权系数)
*   **系数**：
    *   高商业属性 (0.3)：广告、商业片、带货内容。
    *   中商业属性 (0.2)：网剧、通俗小说、综艺。
    *   低商业属性 (0.1)：文艺片、纪录片、公益内容。

### 4.3 附加与扣分
*   **附加分 (Max +3)**：原创突破、正向价值、行业开创性。
*   **扣分项 (Min -5)**：版权侵权、内容违规、虚假宣传。
*   **终审校准 (±5)**：用于修正硬伤、平衡争议、风险校准。

### 4.4 评分等级
*   **卓越标杆级 (150+)**：全维度无短板，行业标杆。
*   **卓越级 (135-149.9)**：顶尖水准，微小瑕疵。
*   **优秀级 (120-134.9)**：上游水准，核心优势突出。
*   **良好级 (105-119.9)**：合格偏上，有亮点，风险可控。
*   **合格级 (90-104.9)**：无致命硬伤，无重大风险。
*   **待完善级 (<90)**：存在核心硬伤或重大风险。
