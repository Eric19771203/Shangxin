# Seedance 2.0 终极创意导演 SkillSet

## 简介
这是 **Seedance 2.0 全能创作系统的终极形态**，深度整合了“专家级创意导演助手”的创意能力、“工业化漫剧生产”的流程规范以及官方手册的多模态技术。无论您是想制作单点爆款短视频，还是生产长篇连载漫剧，本 SkillSet 都能提供从灵感到落地的全流程支持。

## 目录结构
*   **seedance-ultimate-skill/**
    *   `SKILL.md`: 核心定义文件，包含双模式（创意/工业）工作流及指令。
    *   `workflow_guide.md`: **工业化漫剧生产指南**，详解 15秒拆解、全流程一致性锁定、七大衔接策略。
    *   `prompt_library.md`: **终极提示词库**，包含 9大场景模板、创意复刻、特效迁移、视频编辑等标准化 Prompt。
    *   `best_practices.md`: **最佳实践与避坑**，涵盖物理模拟约束、动作描述技巧、参数设置建议。

## 核心功能
1.  **双模式驱动**：
    *   **创意模式**：单点生成电商 TVC、特效转场、音乐 MV。
    *   **工业模式**：批量生产长篇漫剧，保证角色不崩、画风不飘。
2.  **一致性解决方案**：
    *   **尾帧锚定**：用上一段结尾生成下一段开头。
    *   **全局参考池**：锁定角色三视图和场景全景图。
    *   **音频先导**：利用 J-cut 声音提前进入掩盖视觉跳变。
3.  **多模态精准控制**：支持 `@图片`、`@视频`、`@音频` 混合引用，实现精准的运镜复刻和风格迁移。

## 快速开始
*   **单点创作**：`/create [描述]`（如“复刻这个视频的运镜”）
*   **长篇生产**：`/series [剧本]`（启动 15秒模块化拆解流程）
*   **获取模板**：`/template [场景]`（如“古装预告”）
*   **查看避坑**：`/fix [问题]`（如“角色崩脸怎么办”）
