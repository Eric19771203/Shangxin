# 终极提示词库 (Ultimate Prompt Library)

本库整合了“专家级创意导演”的场景模板与官方手册的高级玩法，所有 Prompt 均遵循 **5步万能公式**：
`素材引用 (@) + 核心目标 + 镜头语言 + 时间线描述 + 风格与约束`

## 1. 9大高频场景模板 (Standard Scenes)

### 场景1：电商产品 TVC
*   **Prompt**: `使用 @产品图 作为主体。核心目标：展现产品高级质感与细节。镜头语言：微距特写开场，缓慢拉远展示全貌，光影扫过表面。0-3s：微距展示材质纹理；3-10s：产品功能动态演示；10-15s：品牌Logo定格。风格：明亮商业布光，4K超清，无噪点。约束：保持产品结构不变形。`

### 场景2：古装/短剧预告
*   **Prompt**: `使用 @角色图 作为主角。核心目标：展现虐心剧情张力。镜头语言：面部特写，背景虚化，缓慢推镜。0-5s：主角眼含热泪，神情悲伤；5-10s：大雨落下，情绪爆发，抬头望天；10-15s：背影远去，留白悬念。风格：古风电影感，冷色调，雨天氛围。约束：古装形制准确，口型自然。`

### 场景3：动作/格斗 (物理模拟)
*   **Prompt**: `使用 @角色A @角色B。核心目标：展现拳拳到肉的打击感。镜头语言：手持跟拍，动态模糊，冲击力构图。0-5s：双方对峙，气场流动；5-12s：快速交手，物理碰撞产生空气波纹；12-15s：一方倒地，特写胜利者。风格：高对比度，硬朗动作片质感。约束：动作符合人体工学，无穿模。`

### 场景4：音乐卡点/MV
*   **Prompt**: `使用 @素材集。核心目标：画面随 @音乐 节奏律动。镜头语言：快速剪辑，画面缩放。0-15s：每逢鼓点重音，画面进行一次快速切换或光影闪烁，色彩随情绪爆发。风格：赛博朋克霓虹色，动感强烈。约束：音画严格同步。`

## 2. 创意复刻与特效迁移 (Advanced Creative)

### 2.1 风格化复刻 (Style Transfer)
*   **Prompt**: `[风格描述]（如黑白水墨/像素风），@图片1 的人物参考 @视频1 的特效和动作，上演一段 [具体动作]，保留原视频的运镜节奏，将材质替换为水墨粒子消散效果。`

### 2.2 运镜模仿 (Camera Copy)
*   **Prompt**: `参考 @视频1 的希区柯克变焦运镜。@图片1 的主体保持大小不变，背景极速后退，制造强烈的空间压缩感，渲染惊悚/悬疑氛围。`

### 2.3 剧情颠覆/编辑 (Plot Twist)
*   **Prompt**: `颠覆 @视频1 的剧情。原视频中 [角色] 微笑，现在改为 [新表情]（如冷笑），在 [时间点] 突然 [新动作]（如推倒对方）。动作干脆利落，彻底改变故事走向。`

## 3. 视频延长与补全 (Extension)

### 3.1 视频延长 (Extension)
*   **Prompt**: `将 @视频1 延长 6秒。保持原有光影和运镜速度，镜头继续向右平移，展现 [新出现的物体/场景]，最后自然淡出至黑屏。`

### 3.2 静态图转视频 (Image to Video)
*   **Prompt**: `基于 @图片1。0-3s：画面静止，仅有微风吹动衣角；3-10s：人物缓慢转头，眼神看向镜头，露出微笑；10-15s：伸手邀请，镜头推进。风格：治愈系日漫风。`
