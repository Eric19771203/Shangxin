# 工业化漫剧生产流程指南 (Industrial Workflow Guide)

## 1. 核心理念：模块化与一致性
将长视频生产拆解为可控的 **15秒标准化模块**，并通过多重锚定技术确保模块间的无缝衔接。

## 2. 第一步：15秒分镜模块化拆解
Seedance 2.0 的黄金生成周期为 15秒。所有长剧本必须按此标准拆解。

### 标准拆分公式
| 时间区间 | 核心功能 | 镜头规则 | 生成约束 |
| :--- | :--- | :--- | :--- |
| **0-3秒** | **钩子镜头** | 1个固定特写/强冲突画面 | 单镜单动作，锁定核心锚点 |
| **3-10秒** | **主体叙事** | 2-3个镜头，全景→中景→近景递进 | 单一主运镜，避免混乱 |
| **10-15秒** | **收尾衔接** | 1个稳定结尾镜头，留出动作余量 | **结尾帧必须清晰稳定**，作为下一段参考 |

### 自动化规则
1.  **粒度约束**：单镜时长控制在 2-5秒。
2.  **完整性**：每个 15秒模块必须包含完整的“起承转合”微弧。
3.  **标准化输出**：`[模块ID] 15s | 核心剧情 | 镜头序列 | 统一设定`

## 3. 第二步：全流程一致性锁定 (Anti-Drift Strategy)
解决长视频“崩脸”和“画风漂移”的组合拳方案。

### 方案 A：全局参考池 (Global Reference Pool) - **基础必选**
*   **操作**：项目启动前，建立角色三视图、主场景全景图、画风参考图。
*   **执行**：所有模块 Prompt 开头强制植入：
    > `@全局角色ID @全局场景ID @全局画风ID 严格保持所有设定与参考图完全一致，全片统一`

### 方案 B：尾帧锚定 (Tail-Frame Anchoring) - **同场景衔接**
*   **操作**：提取上一段生成的 **最后一帧**。
*   **执行**：下一段 Prompt 引用：
    > `@上一段尾帧ID 作为核心参考，承接上一镜头动作，连贯流畅`

### 方案 C：音频先导 (Audio Lead-in) - **隐形缝合**
*   **操作**：下一段的配音/音效提前 0.5-1秒 进入上一段结尾 (J-cut)。
*   **效果**：利用听觉连贯性掩盖视觉微跳变。

## 4. 第三步：七大衔接策略 (Transition Strategies)

| 场景 | 推荐策略 | 提示词要点 |
| :--- | :--- | :--- |
| **剧情转折/换场** | **断点式转折** | 结尾用空镜/特写，利用叙事断点天然切换 |
| **同场景对话** | **尾帧锚定** | 引用上一帧，保持站位和光影 |
| **激烈打斗** | **重叠帧叠化** | 前后各多生成 2秒重叠动作，后期做交叉溶解 |
| **蒙太奇/快穿** | **匹配剪辑** | 前后镜头在构图/形状/色彩上强制匹配 |
| **长镜头跟拍** | **原生多镜头** | 一次性输入多段分镜，利用 Seedance 原生一致性 |

## 5. 生产避坑红线
1.  **禁止** 单个 15秒内使用超过 2 种复杂运镜。
2.  **禁止** 前后模块相机参数（焦距/视角）突变。
3.  **禁止** 参考图权重超过 0.8（会锁死动作）。
4.  **禁止** 随意更改生成参数（步数/CFG/画风强度）。
