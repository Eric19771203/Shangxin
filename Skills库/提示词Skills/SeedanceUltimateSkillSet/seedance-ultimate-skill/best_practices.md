# 最佳实践与避坑指南 (Best Practices)

## 1. 提示词编写技巧
*   **动作描述**：
    *   ✅ **推荐**：缓慢、轻柔、连贯、自然、流畅。
    *   ❌ **避免**：剧烈、复杂、多人互动、大幅度扭曲。
    *   *Seedance 2.0 最擅长“慢动作”和“微表情”，过于剧烈的动作容易崩坏。*
*   **镜头语言**：
    *   ✅ **推荐**：缓慢推镜、平稳跟拍、固定镜头。
    *   ❌ **避免**：极速甩镜、无规则晃动。
*   **画质增强**：
    *   通用词：`4K, 超高清, 电影质感, 细节丰富, 无噪点, 无闪烁`。

## 2. 物理模拟与约束
*   **重力/流体**：在涉及水、布料、头发时，加入 `符合物理重力, 自然飘动`。
*   **碰撞检测**：动作戏必须加 `物理碰撞真实, 无穿模, 人体结构正常`。

## 3. 常见问题解决方案 (Troubleshooting)

| 问题现象 | 可能原因 | 解决方案 |
| :--- | :--- | :--- |
| **角色崩脸** | 动作幅度过大 / 未绑定参考图 | 减小动作幅度，使用 `@尾帧` 或 `@全局参考` 锁定面部 |
| **画风漂移** | 提示词中风格描述不一致 | 检查前后 Prompt 的风格词，使用 `@全局画风` 锁定 |
| **画面闪烁** | 细节过多 / 采样步数不足 | 简化背景描述，增加 `无闪烁, 画面稳定` 负面提示 |
| **动作僵硬** | 参考图权重过高 | 将参考图权重降至 0.6-0.7，给 AI 更多发挥空间 |
| **过渡生硬** | 镜头切换太快 | 使用“断点式转折”或“匹配剪辑”策略，不要在单镜头内硬切 |

## 4. 参数设置建议
*   **视频延长**：时长代表 **新增时长**（如原视频 5s，延长 5s = 总长 10s）。
*   **抽卡策略**：建议单次生成 **4条**，从中择优，因为 AI 生成具有随机性。
*   **参考权重**：
    *   强一致性需求（如电商品）：0.8-0.9
    *   动作幅度大（如舞蹈）：0.5-0.7

## 5. 禁忌红线 (Don'ts)
*   **不写** 色情、暴力、版权敏感人物。
*   **不写** 逻辑矛盾的指令（如“极速奔跑但画面静止”）。
*   **不写** 过于抽象的形容词（如“好看”、“精彩”），要写具体的视觉元素。
