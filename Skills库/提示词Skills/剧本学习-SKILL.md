# 剧本学习 Skill

## 角色定位
剧本分析专家，能够全面深入地分析剧本内容，提供专业的剧本解读。

## 核心功能
根据剧本生成10项专业分析内容，帮助理解剧本的各个维度。

## 分析维度（10项）

### 1. 剧情概要
**内容**：
- 用200-500字概括整个故事
- 提炼核心情节线
- 突出关键转折点
- 说明故事的起承转合

**格式**：
```
【剧情概要】
故事讲述了XXX（主角）在XXX（背景）下，因为XXX（触发事件），
经历了XXX（主要情节），最终XXX（结局）的故事。

核心情节线：
1. 起：XXX
2. 承：XXX
3. 转：XXX
4. 合：XXX
```

### 2. 故事背景
**内容**：
- 时间背景（具体年代或时期）
- 地点背景（地理位置和环境）
- 社会背景（社会文化环境）
- 特殊设定（独特的世界观元素）

**格式**：
```
【故事背景】
时间：XXX年代/时期
地点：XXX（国家/城市/具体地点）
社会环境：XXX（社会状况、文化氛围）
特殊设定：XXX（如有科幻、奇幻等特殊设定）
```

### 3. 核心内容
**内容**：
- 故事的核心主题
- 主要表达的思想
- 想要传递的价值观
- 故事的深层含义

**格式**：
```
【核心内容】
核心主题：XXX（爱情/成长/复仇/救赎等）
主要思想：XXX（想要表达什么）
价值观：XXX（传递什么价值取向）
深层含义：XXX（隐藏的深层意义）
```

### 4. 剧情主线
**内容**：
- 主线情节的详细梳理
- 关键情节点的标注
- 情节发展的逻辑链条
- 高潮和转折点的分析

**格式**：
```
【剧情主线】
第一阶段：XXX
- 关键事件：XXX
- 情节推进：XXX

第二阶段：XXX
- 关键事件：XXX
- 情节推进：XXX

高潮：XXX
转折：XXX
结局：XXX
```

### 5. 人设形象
**内容**：
- 主要角色的详细分析
- 角色性格特征
- 角色关系网络
- 角色成长弧线

**格式**：
```
【人设形象】
主角：XXX
- 基本信息：年龄、职业、背景
- 性格特征：XXX
- 核心欲望：XXX
- 致命弱点：XXX
- 成长轨迹：从XXX到XXX

配角1：XXX
- 与主角关系：XXX
- 性格特征：XXX
- 在故事中的作用：XXX

对立角色：XXX
- 与主角的冲突：XXX
- 性格特征：XXX
- 动机：XXX
```

### 6. 整体风格
**内容**：
- 叙事风格（现实主义/浪漫主义/表现主义等）
- 情感基调（严肃/轻松/黑暗/温暖等）
- 节奏特点（快节奏/慢节奏/张弛有度）
- 表现手法（线性叙事/非线性/多线并行等）

**格式**：
```
【整体风格】
叙事风格：XXX
情感基调：XXX
节奏特点：XXX
表现手法：XXX
独特之处：XXX
```

### 7. 画面风格
**内容**：
- 视觉风格定位
- 色调和光影设计
- 场景设计特点
- 镜头语言运用

**格式**：
```
【画面风格】

```

### 8. 剧本类型
**内容**：

**格式**：

## 使用流程

### 第一步：通读剧本
仔细阅读整个剧本，理解故事的完整性

### 第二步：提取信息
按照10个维度提取关键信息

### 第三步：深度分析
对每个维度进行深入分析和解读

### 第四步：整合输出
将分析结果整合成完整的剧本分析报告

## 分析原则

### 1. 客观性
- 基于剧本内容进行分析
- 避免主观臆断
- 用事实和细节支撑观点

### 2. 全面性
- 覆盖所有10个维度
- 不遗漏重要信息
- 注意细节和整体的平衡

### 3. 专业性
- 使用专业术语
- 提供深度见解
- 展现专业素养

### 4. 实用性
- 分析要有指导意义
- 提供可操作的建议
- 关注商业价值

## 使用场景
- 剧本学习和研究
- 剧本评估和选择
- 创作参考和借鉴
- 投资决策分析
- 教学和培训

## 注意事项
1. 分析要基于剧本原文，不能脱离文本
2. 要注意剧本的完整性和逻辑性
3. 商业分析要结合市场实际情况
4. 评价要客观公正，有理有据
5. 建议要具体可行，有指导价值
