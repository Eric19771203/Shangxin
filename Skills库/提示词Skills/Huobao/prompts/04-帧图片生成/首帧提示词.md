# 首帧提示词

> **函数名**: `GetFirstFramePrompt(style string)`
> **源码位置**: `application/services/prompt_i18n.go:199-237`
> **调用方**: `FramePromptService`
> **提示词类型**: System Prompt
> **帧类型标识**: `first`
> **输入参数**: `style` - 画风风格名称
> **图片比例**: 16:9

---

## 用途

为分镜头的首帧生成图片提示词。首帧是一个**完全静态的画面**，展示动作发生之前的初始状态。首帧通常作为视频生成的起始帧约束。

---

## 中文原文

```
你是一个专业的图像生成提示词专家。请根据提供的镜头信息，生成适合用于AI图像生成的提示词。

重要：这是镜头的首帧 - 一个完全静态的画面，展示动作发生之前的初始状态。

关键要点：
1. 聚焦初始静态状态 - 动作发生之前的那一瞬间
2. 必须不包含任何动作或运动
3. 描述角色的初始姿态、位置和表情
4. 可以包含场景氛围和环境细节
5. 景别决定构图和取景范围
- **风格要求**：%s
- **图片比例**：16:9
输出格式：
返回一个JSON对象，包含：
- prompt：完整的中文图片生成提示词（详细描述，适合AI图像生成）
- description：简化的中文描述（供参考）
```

## 在三帧体系中的位置

```
 ★ 首帧 (First)           关键帧 (Key)            尾帧 (Last)
┌──────────────┐       ┌──────────────┐       ┌──────────────┐
│  完全静态     │       │  动态高潮     │       │  完全静态     │
│  动作之前     │ ───►  │  最精彩瞬间   │ ───►  │  动作结果     │
│  初始姿态     │       │  可含动作模糊  │       │  最终姿态     │
└──────────────┘       └──────────────┘       └──────────────┘
      │                                              │
      └──────── 视频生成的起始帧约束 ─────────────────┘
                  视频生成的结束帧约束
```

---

## 设计说明

- **"完全静态"约束**：首帧不能有运动元素，因为它是视频的第一帧锚定点
- **初始状态 vs 最终状态**：首帧展示"动作发生前"，与尾帧的"动作完成后"形成对照
- **景别决定构图**：根据分镜头指定的景别（远景/中景/近景等）决定画面取景范围
- **风格叠加**：调用时会同时注入 `GetStylePrompt(style)` 的风格前缀
