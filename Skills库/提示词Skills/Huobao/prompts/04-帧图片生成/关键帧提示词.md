# 关键帧提示词

> **函数名**: `GetKeyFramePrompt(style string)`
> **源码位置**: `application/services/prompt_i18n.go:239-277`
> **调用方**: `FramePromptService`
> **提示词类型**: System Prompt
> **帧类型标识**: `key`
> **输入参数**: `style` - 画风风格名称
> **图片比例**: 16:9

---

## 用途

为分镜头的关键帧生成图片提示词。关键帧捕捉的是**动作最激烈、最精彩的瞬间**，是镜头的视觉高潮点。

---

## 中文原文

```
你是一个专业的图像生成提示词专家。请根据提供的镜头信息，生成适合用于AI图像生成的提示词。

重要：这是镜头的关键帧 - 捕捉动作最激烈、最精彩的瞬间。

关键要点：
1. 聚焦动作最精彩的时刻
2. 捕捉情绪表达的顶点
3. 强调动态张力
4. 展示角色动作和表情的高潮状态
5. 可以包含动作模糊或动态效果
- **风格要求**：%s
- **图片比例**：16:9
输出格式：
返回一个JSON对象，包含：
- prompt：完整的中文图片生成提示词（详细描述，适合AI图像生成）
- description：简化的中文描述（供参考）
```

## 与首帧、尾帧的区别

| 对比项 | 首帧 | 关键帧 | 尾帧 |
|--------|------|--------|------|
| 画面状态 | 完全静态 | 可含动态效果 | 完全静态 |
| 时间点 | 动作之前 | 动作高潮 | 动作之后 |
| 情绪 | 初始情绪 | 情绪顶点 | 情绪余韵 |
| 特效 | 无 | 动作模糊/动态效果 | 无 |
| 视频用途 | 起始帧 | 中间参考 | 结束帧 |

---

## 设计说明

- **"动作模糊"是关键帧独有的**：首帧和尾帧都是静态的，只有关键帧允许包含运动感
- **情绪顶点**：关键帧应该展现分镜头的情感高峰，与情绪强度标记（emotion_intensity）对应
- **独立展示价值**：关键帧本身就是最具视觉冲击力的画面，可单独用于缩略图/海报
