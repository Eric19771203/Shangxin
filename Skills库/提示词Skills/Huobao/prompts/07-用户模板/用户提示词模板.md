# FormatUserPrompt 用户提示词模板

> **函数名**: `FormatUserPrompt(key string, args ...interface{})`
> **源码位置**: `application/services/prompt_i18n.go:591-673`
> **提示词类型**: User Prompt 片段（与 System Prompt 配合使用）

---

## 用途

`FormatUserPrompt` 是一个通用的模板函数，根据 `key` 和动态参数 `args` 生成格式化的用户提示词片段。这些片段与 System Prompt 配合，组成完整的 AI 调用请求。

---

## 完整模板列表

### 一、剧本创作类模板

#### `outline_request` — 大纲生成请求

**中文**:
```
请为以下主题创作短剧大纲：

主题：%s
```

**英文**:
```
Please create a short drama outline for the following theme:

Theme: %s
```

**参数**: `args[0]` = 用户输入的主题文本

---

#### `genre_preference` — 类型偏好

**中文**: `\n类型偏好：%s`
**英文**: `\nGenre preference: %s`

**参数**: `args[0]` = 类型名称（如"悬疑"、"爱情"等）

---

#### `style_requirement` — 风格要求

**中文**: `\n风格要求：%s`
**英文**: `\nStyle requirement: %s`

**参数**: `args[0]` = 风格名称

---

#### `episode_count` — 集数

**中文**: `\n剧集数量：%d集`
**英文**: `\nNumber of episodes: %d episodes`

**参数**: `args[0]` = 集数（整数）

---

#### `episode_importance` — 集数完整性约束

**中文**:
```
**重要：必须在episodes数组中规划完整的%d集剧情，每集都要有明确的故事内容！**
```

**英文**:
```
**Important: Must plan complete storylines for all %d episodes in the episodes array, each with clear story content!**
```

**参数**: `args[0]` = 集数（整数）

---

### 二、角色与剧本类模板

#### `character_request` — 角色提取请求

**中文**:
```
剧本内容：
%s

请从剧本中提取并整理最多 %d 个主要角色的详细设定。
```

**英文**:
```
Script content:
%s

Please extract and organize detailed character profiles for up to %d main characters from the script.
```

**参数**: `args[0]` = 剧本全文, `args[1]` = 最大角色数量

---

#### `episode_script_request` — 分集剧本生成请求

**中文**:
```
剧本大纲：
%s
%s
请基于以上大纲和角色，创作 %d 集的详细剧本。

**重要要求：**
- 必须生成完整的 %d 集，从第1集到第%d集，不能遗漏
- 每集约3-5分钟（150-300秒）
- 每集的duration字段要根据剧本内容长度合理设置，不要都设置为同一个值
- 返回的JSON中episodes数组必须包含 %d 个元素
```

**英文**:
```
Drama outline:
%s
%s
Please create detailed scripts for %d episodes based on the above outline and characters.

**Important requirements:**
- Must generate all %d episodes, from episode 1 to episode %d, cannot skip any
- Each episode is about 3-5 minutes (150-300 seconds)
- The duration field for each episode should be set reasonably based on script content length, not all the same value
- The episodes array in the returned JSON must contain %d elements
```

**参数**: `args[0]` = 大纲JSON, `args[1]` = 角色设定, `args[2-5]` = 集数（重复用于不同位置）

## 组装示例

以下展示一个完整的帧提示词 User Prompt 组装过程：

```go
// 1. 组装镜头上下文
context := ""
context += p.FormatUserPrompt("shot_description_label", storyboard.Title)
context += "\n" + p.FormatUserPrompt("scene_label", storyboard.Location, storyboard.Time)
context += "\n" + p.FormatUserPrompt("characters_label", characterNames)
context += "\n" + p.FormatUserPrompt("action_label", storyboard.Action)
context += "\n" + p.FormatUserPrompt("result_label", storyboard.Result)
context += "\n" + p.FormatUserPrompt("dialogue_label", storyboard.Dialogue)
context += "\n" + p.FormatUserPrompt("atmosphere_label", storyboard.Atmosphere)
context += "\n" + p.FormatUserPrompt("shot_type_label", storyboard.ShotType)
context += "\n" + p.FormatUserPrompt("angle_label", storyboard.Angle)
context += "\n" + p.FormatUserPrompt("movement_label", storyboard.Movement)

// 2. 包装为帧请求
userPrompt := p.FormatUserPrompt("frame_info", context)

// 3. 获取系统提示词
systemPrompt := p.GetFirstFramePrompt(style)

// 4. 叠加风格提示词
stylePrompt := p.GetStylePrompt(style)
fullSystemPrompt := stylePrompt + "\n\n" + systemPrompt

// 5. 调用 AI
result := aiService.GenerateText(userPrompt, fullSystemPrompt)
```

最终组装的 User Prompt 如下：

```
镜头信息：
镜头描述: 陈晨突然抬头，眼神震惊
场景: 豪华办公室内, 下午
角色: 陈晨
动作: 陈晨坐在办公桌前翻阅文件时突然抬头
结果: 陈晨面部特写，表情从专注变为震惊
对白: 这不可能...
氛围: 从平静到紧张的氛围突变
景别: 近景
角度: 平视
运镜: 推镜

请直接生成首帧的图像提示词，不要任何解释：
```
