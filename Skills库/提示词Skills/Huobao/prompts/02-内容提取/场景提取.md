# 场景提取提示词

> **函数名**: `GetSceneExtractionPrompt(style string)`
> **源码位置**: `application/services/prompt_i18n.go:146-197`
> **调用方**: `ImageGenerationService.extractBackgroundsFromScript()`
> **提示词类型**: System Prompt
> **输入参数**: `style` - 画风风格名称
> **图片比例**: 16:9

---

## 用途

从剧本中提取所有唯一的场景背景（地点+时间组合），为每个场景生成**纯背景（无人物）**的图片生成提示词。

---

## 中文原文

```
【任务】从剧本中提取所有唯一的场景背景

【要求】
1. 识别剧本中所有不同的场景（地点+时间组合）
2. 为每个场景生成详细的**中文**图片生成提示词（Prompt）
3. **重要**：场景描述必须是**纯背景**，不能包含人物、角色、动作等元素
4. Prompt要求：
   - **必须使用中文**，不能包含英文字符
   - 详细描述场景、时间、氛围、风格
   - 必须明确说明"无人物、无角色、空场景"
   - 要符合剧本的题材和氛围
   - **风格要求**：%s
   - **图片比例**：16:9

【输出格式】
**重要：必须只返回纯JSON数组，不要包含任何markdown代码块、说明文字或其他内容。直接以 [ 开头，以 ] 结尾。**

每个元素包含：
- location：地点（如"豪华办公室"）
- time：时间（如"下午"）
- prompt：完整的中文图片生成提示词（纯背景，明确说明无人物）
```

## 设计说明

- **纯背景约束**：场景图后续会作为背景图层，与角色图层合成，所以必须无人物
- **"无人物、无角色、空场景"**：三重否定确保 AI 图片模型不会在背景中画人
- **地点+时间去重**：同一地点不同时间段算不同场景（如"办公室·白天"和"办公室·夜晚"）
- **中文/英文语言分离**：中文模式生成中文提示词，英文模式生成英文提示词，适配不同的图片生成模型
- **风格参数**：注入画风风格，确保所有场景图风格一致
