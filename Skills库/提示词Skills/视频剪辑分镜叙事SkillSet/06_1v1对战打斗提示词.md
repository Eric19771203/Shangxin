# 06_1v1对战打斗提示词 (1v1 Fight Scene Prompts)

## 核心功能
外网爆款的1v1对战/打斗专用提示词，适用于需要生成高动态、流畅连贯战斗场景的AI视频生成。

## 1v1 Fight Scene Prompt (English)
```
Create a dynamic 1v1 combat sequence with smooth continuity.
Two characters fighting in an intense battle.
Include:
- Fast cuts between close-ups and wide shots
- Impact frames with motion blur
- Dynamic camera movement: tracking, dolly, whip pan
- Reaction shots after each hit
- Climax with a powerful finishing move
- Clear choreography, no messy frames
- Cinematic lighting, high contrast, sharp action
- Fluid motion, impact lines, dynamic poses
- Tight rhythm: attack → block → dodge → counter

Style: action anime, cinematic fight, high energy, smooth animation.
```

## 1v1 对战打斗提示词 (Chinese)
```
创作一段流畅连贯的1v1对战打斗镜头。
两名角色进行激烈对决，包含：
- 快切特写与全景交替
- 带运动模糊的打击帧
- 动态运镜：跟拍、推进、甩镜
- 每次攻击后的反应镜头
- 带有强力终结技的高潮
- 动作清晰，不杂乱
- 电影感光影，高对比
- 流畅动作、冲击感、动态姿势
- 紧凑节奏：攻击→格挡→闪避→反击

风格：高能动作、电影级打斗、流畅连贯。
```
