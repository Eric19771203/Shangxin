# 08_Skills技能库大全 (Video & Storyboard Skills Library)

## 核心功能
提供现成、可直接塞进提示词的「Skills 技能库」，涵盖分镜、打斗、三视图、叙事、剪辑等固定句式。

## 1. 分镜 / 叙事 Skills（必加）
```
- Perfect shot composition
- Rule of thirds
- Dynamic camera movement
- Clear visual storytelling
- Smooth scene transition
- Consistent emotional rhythm
- Front/side/back view switching
- Close-up & wide shot alternation
- Tension building through editing
- Cinematic sequence structure
```

## 2. 1v1 打斗 / 战斗 Skills（最强）
```
- Professional combat choreography
- Fast and smooth action flow
- Impact frames with motion blur
- Attack → block → dodge → counter
- Dynamic fighting poses
- High impact hit effects
- Realistic weight and force
- Fluid character movement
- Intense battle rhythm
- Clear fight logic
```

## 3. 三视图 / 设定图 Skills（究极稳）
```
- Orthographic projection
- No perspective distortion
- Perfect symmetry
- Front/side/back consistent
- Same height & proportion
- Clean technical drawing
- Neat layout in one image
- Professional design sheet
- High detail line art
- Official reference quality
```

## 4. 电影感 / 剪辑节奏 Skills
```
- Cinematic lighting
- High contrast shadows
- Smooth pacing control
- Visual impact in every frame
- Clear story structure
- Hook at the beginning
- Climax at the middle
- Emotional ending
- Professional color grading
- 8K ultra detailed
```

## 5. 25宫格 / 50宫格连续叙事 Skills
```
- 25-shot complete storyboard
- 50-shot full narrative sequence
- Setup → conflict → climax → resolution
- Continuous visual flow
- Varied shot types
- No repeated composition
- Logical scene arrangement
- High continuity
```

## 📦 究极整合版 Skills (Ultimate Integrated Skills)
你贴进任何提示词末尾都超强：
```
skills: perfect composition, dynamic camera, smooth action flow, orthographic three views, no distortion, professional combat choreography, clear narrative, cinematic lighting, high detail, 8K
```
