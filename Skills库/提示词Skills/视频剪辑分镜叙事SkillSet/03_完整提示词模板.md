# 03_完整提示词模板 (Complete Prompt Templates)

## 核心功能
提供组合好的Prompt句式，方便直接替换关键词使用。

## 模板1：电影感单镜头 (Cinematic Single Shot)

**格式：**
```
[Shot type] of [subject] [action] in [setting],
[Camera movement] with [lens],
[Lighting] + [color grade],
[Mood/tone], [duration], [aspect ratio]
```

**示例：**
```
Medium close-up of detective examining a clue in a dim office,
Slow dolly in with 85mm lens,
Low-key chiaroscuro lighting, cool blue grade,
Suspenseful, 8s, 2.39:1
```

## 模板2：分镜序列 (Storyboard Sequence)

**格式：**
```
镜1：[景别] [运镜] [画面]，时长X秒，情绪：[X]
镜2：[景别] [运镜] [画面]，时长X秒，情绪：[X]
镜3：[景别] [运镜] [画面]，时长X秒，情绪：[X]
转场：[匹配剪辑/淡入淡出/切]
节奏：[慢/中/快]
```

**示例：**
```
镜1：全景 慢推 雨夜街道，4秒，压抑
镜2：中近景 跟拍 主角奔跑，3秒，紧张
镜3：特写 定格 主角惊恐眼神，2秒，惊悚
转场：匹配剪辑（脚步→心跳）
节奏：快
```
