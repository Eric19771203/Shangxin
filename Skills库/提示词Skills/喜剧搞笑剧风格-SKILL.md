# 喜剧搞笑剧风格

## 工具说明
提供轻喜剧等喜剧搞笑题材的完整风格提示词模板

## 核心功能
- 提供喜剧搞笑细分类型
- 包含视觉风格、色调、光照、场景提示词
- 中英文双语提示词模板

## 使用场景
- 轻喜剧
- 家庭喜剧
- 搞笑短剧

## 提示词模板

### 中文模板
```
[选择一种类型]

轻喜剧(3D)：
Pixar风格，3D动漫，可爱角色设计，多彩，有趣表情，家庭友好，明亮光照，欢快氛围

轻喜剧(2D)：
Q版风格，可爱动漫，超变形，卡哇伊，明亮色彩，喜剧，简单背景，有趣
```
