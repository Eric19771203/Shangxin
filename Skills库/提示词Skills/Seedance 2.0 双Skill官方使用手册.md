|常见问题|官方解决方案|
|---|---|

3. **高手级**：@多素材 + 分镜秒数 + 精确运镜 + 光影情绪

### 1.4 官方规则速查


## 第二章：Skill 2 - Seedance 2.0 官方提示词自动生成 Skill

### 2.1 技能定位与适用场景

|项目|详情|
|---|---|
|技能定位|基于官方规则的自动化提示词生成工具，直接输出可使用的专业提示词|
|适用人群|所有 Seedance 2.0 用户，尤其是不会写提示词、想要快速生成内容的用户|
|适用场景|快速生成提示词、优化现有提示词、生成专项场景提示词（视频延长、角色替换等）|
|核心价值|100% 遵循官方规则，零合规风险，直接复制使用，大幅提升创作效率|
### 2.2 触发方式

当你出现以下行为时，该 Skill 会自动触发：

1. 输入 Seedance 2.0 的视频创作需求，需要生成对应提示词

2. 提供分镜脚本、创作想法，需要转化为平台适配的标准提示词

3. 调用 Skill 1 后，需要生成对应提示词

4. 需要优化现有提示词，提升生成效果

5. 需要生成专项场景提示词（视频延长、角色替换、特效复刻等）

### 2.3 输入要求与生成流程

#### 2.3.1 极简输入项（仅需填写以下内容）

1. **核心主体**：人物 / 产品 / 核心主角（如双女主宫廷打斗、XX 漫画手办）

2. **参考素材**：@图片 /@视频 /@音频的数量与用途（无则填 "无"）

3. **固定场景**：视频发生的环境（如深夜皇宫鎏金回廊、潮玩手办展厅）

4. **总时长**：4-15 秒（平台上限）

5. **镜头数量**：如 1 个长镜头 / 5-8 个分镜

6. **核心风格**：如动漫武侠写实、3D 潮玩手办质感、电影级广告风

#### 2.3.2 自动生成流程

1. 接收你的输入内容，同步调用 Skill 1 进行规则校验

2. 按照官方黄金公式自动填充提示词全模块内容

3. 自动分配单镜头时长、景别、运镜、动作，形成完整分镜时间轴

4. 自动匹配官方运镜写法、风格词、音效、负面提示

5. 自动完成合规校验，规避写实人脸、敏感内容、参数超限等风险

6. 同步输出「全能参考入口完整版」和「首尾帧入口分镜版」两套提示词

### 5.2 平台参数限制

- 图片输入：≤9 张，单张小于 30MB，格式支持 jpeg、png、webp、bmp、tiff、gif

- 视频输入：≤3 个，总时长 2-15s，单个小于 50MB，分辨率 480p-720p，格式支持 mp4、mov

- 音频输入：≤3 个，总时长≤15s，单个小于 15MB，格式支持 mp3、wav

- 混合输入：总文件≤12 个，符合上述单类型格式要求

- 生成时长：4-15s，单段最长 15 秒，可后期拼接延长成片

### 5.3 合规红线

- 禁止上传包含清晰可辨识的写实真人脸部的图片 / 视频素材

- 禁止生成深假、违规、侵权、涉敏内容

- 禁止提交过长、涉敏感内容的提示词
