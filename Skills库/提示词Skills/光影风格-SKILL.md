# 光影风格

## 工具说明
提供高对比度、柔光风格、体积光三种核心光影选择，根据剧集类型营造戏剧张力或温柔氛围

## 核心功能
- 提供三种核心光影风格
- 包含光影特点和关键词
- 推荐适合的剧集类型

## 使用场景
- 悬疑、惊悚、黑帮、史诗（高对比度）
- 爱情、治愈、日常、童话（柔光风格）
- 奇幻、史诗、宗教、魔法（体积光）

## 细分光影风格

### 高对比度 (High Contrast)
```
特点：
- 明暗分明
- 戏剧张力强
- 电影感强烈

关键词：
dramatic lighting, high contrast,
chiaroscuro, deep shadows,
film noir style

适合：悬疑、惊悚、黑帮、史诗
```

### 柔光风格 (Soft Light)
```
特点：
- 柔和过渡
- 温柔舒适
- 梦幻感

关键词：
soft lighting, diffused light,
gentle shadows, dreamy atmosphere,
ethereal glow

适合：爱情、治愈、日常、童话
```

### 体积光 (Volumetric Lighting)
```
特点：
- 光束可见
- 空间感强
- 神圣/史诗感

关键词：
volumetric lighting, god rays,
light shafts, atmospheric perspective,
cinematic rays

适合：奇幻、史诗、宗教、魔法
```

## 参数说明

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 高对比度 | 戏剧张力 | dramatic lighting, high contrast |
| 柔光风格 | 温柔梦幻 | soft lighting, diffused light |
| 体积光 | 史诗神圣 | volumetric lighting, god rays |
