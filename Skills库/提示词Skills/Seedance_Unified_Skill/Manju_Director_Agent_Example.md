# 漫剧导演 Agent 示例 (Manju Director Agent)

本文件演示如何创建一个**「上级 Agent」**（Director Agent）来管理 `Seedance 2.0 终极全能创作大师 Skill`。

## Agent 角色定义
*   **名称**：漫剧总导演 (Manju Director)
*   **职责**：全流程把控漫剧生产，负责将“故事剧本”转化为“成片视频”。
*   **核心能力**：
    1.  **剧本拆解**：调用《漫剧剧本创作 SkillSet》。
    2.  **分镜提示词生成**：**调用《Seedance 2.0 终极全能创作大师 Skill》**。
    3.  **视频生产与审核**：监督 AI 生成视频，并进行质量把控。

---

## Agent 工作流设计 (Workflow Design)

### 阶段一：剧本与分镜 (Scripting)
1.  **接收需求**：用户输入故事大纲或小说片段。
2.  **调用技能**：《漫剧剧本创作 SkillSet》 -> 生成标准分镜脚本表格。
    *   *输出产物*：包含 `[镜号] [画面描述] [运镜] [时长] [台词]` 的表格。

### 阶段二：提示词批量转化 (Prompt Generation)
**这里是 Seedance Skill 发挥作用的地方：**
1.  **导演指令**："现在激活 `Seedance 2.0 终极全能创作大师 Skill`，对上述分镜脚本表格中的每一个镜头进行处理。"
2.  **执行逻辑**：
    *   遍历每一个分镜行。
    *   提取 `[画面描述]` + `[运镜]` + `[时长]`。
    *   **调用 Skill 的 `高手分镜模板`**，自动填充参数。
    *   **输出**：生成一个对应的 Seedance 提示词列表。

### 阶段三：生产与合成 (Production)
1.  **执行生产**：将生成的提示词复制到 Seedance 2.0 平台生成视频。
2.  **质量检查**：如果生成的视频有问题（如脸崩），导演 Agent 会再次调用 Seedance Skill 的**「故障排查模块」**，生成优化后的提示词重新尝试。
3.  **最终剪辑**：将生成好的视频片段按顺序拼接。

---

## 示例对话演示

**用户**："我想把《红楼梦》黛玉葬花这一段做成漫剧，请开始工作。"

**漫剧总导演 (Agent)**：
"收到。我将按以下步骤执行：

1.  **剧本阶段**：正在调用《漫剧剧本创作 SkillSet》... 已生成 6 个分镜脚本。
2.  **提示词阶段**：正在调用《Seedance 2.0 终极全能创作大师 Skill》...
    *   **镜头 1 (远景)**：`@参考图1...林黛玉肩扛花锄...` (已生成)
    *   **镜头 2 (特写)**：`@参考图1...眼角含泪...` (已生成)
    ... (所有提示词已准备就绪)
3.  **执行阶段**：请将上述提示词复制到即梦平台。如果在生成过程中遇到人物不一致问题，请告诉我，我会调用 Skill 的优化模块为您调整。"

---

## 结论
建立一个 Sub-agent (如漫剧总导演) 是**非常推荐**的，特别是当您需要处理**长流程**或**批量任务**时。它能让 `Seedance Skill` 专注于“生成提示词”这一件事，而把“流程控制”交给上级 Agent，实现更高效的自动化生产。
