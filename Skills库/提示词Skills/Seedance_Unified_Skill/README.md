## 📚 简介
本 Skill 集成了 **Seedance 2.0 官方创作全流程指导** 与 **官方提示词自动生成** 两大核心功能。无论你是新手小白还是资深创作者，都能通过本 Skill 快速掌握 Seedance 2.0 的创作精髓，生成高质量的 AI 视频。

---

## 🚀 核心功能
### 1. 🧠 官方规则咨询师
*   **全量规则查询**：参数限制、合规红线、功能边界，一问便知。
*   **故障排查**：遇到生成失败、画面崩坏等问题，提供官方解决方案。
*   **进阶技巧**：运镜术语、风格词库、多模态玩法教学。

### 2. ⚡ 提示词生成引擎
*   **智能生成**：只需输入一句话需求，自动转化为符合官方黄金公式的专业提示词。
*   **双版本输出**：同时生成适配「全能参考入口」与「首尾帧入口」的两套提示词。
*   **合规校验**：自动规避写实真人脸、敏感词等风险，确保生成成功率。

## 🛠️ 如何使用
### 1. 触发 Skill
在对话框中输入你的需求，例如：
*   "我要生成一段古风视频提示词，主角是穿红衣的女侠"
*   "帮我复刻这个视频的运镜，换成赛博朋克风格"
*   "Seedance 2.0 最长能生成多少秒的视频？"
*   "生成的视频人物脸崩了怎么办？"

### 2. 查看帮助
输入 `/help` 或 `帮助`，获取本 Skill 的详细使用指南。

### 3. 生成提示词
输入 `/generate [需求]`，直接生成提示词。例如：
*   `/generate 一个赛博朋克风格的雨夜街道，霓虹灯闪烁，镜头缓慢推进`
