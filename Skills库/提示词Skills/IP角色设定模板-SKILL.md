# IP角色设定模板

## 工具说明
完整的IP角色设定模板，提供世界观基础设定、角色信息简述、形象气质、内在设定等12个核心维度的设定框架

## 核心功能
- 完整的世界观基础设定（8个维度）
- 全面的角色信息简述
- 详细的形象气质描述
- 深入的内在设定刻画
- 完整的能力体系、身份设定、社会关系
- 丰富的背景故事、使命发展、剧情发展框架

## 使用场景
- IP开发
- 世界观构建
- 角色设定
- 玄幻/科幻作品创作
- 完整人物体系设计
