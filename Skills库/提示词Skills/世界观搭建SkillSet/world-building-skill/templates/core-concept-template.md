# 核心概念模板 (Core Concept Template)

## 1. 世界定调 (World Tone)
*   **名称**：[世界/位面/宇宙的名称]
*   **一句话定调**：[核心矛盾] + [主角身份] + [打破规则]
    *   *示例：异能被财团垄断，底层违禁者少年打破规则。*
*   **题材类型**：[都市异能/东方玄幻/赛博朋克/末世废土...]
*   **视觉风格**：[电影级写实/水墨国风/霓虹赛博/废土机甲...]

## 2. 基础规则 (Fundamental Rules)
*   **时代背景**：[具体年份/纪元，如：新元2077年]
*   **核心法则**：[物理/魔法/科技的底层逻辑，如：能量守恒、等价交换、灵气复苏]
*   **稀缺资源**：[所有人争夺的东西，如：净水、灵石、算力、时间]
*   **核心禁忌**：[绝对不能做的事，如：直视神明、私自改装义体]

## 3. 核心矛盾 (Core Conflict)
*   **表层冲突**：[可见的对抗，如：反抗军 vs 公司狗]
*   **深层矛盾**：[价值观/利益的根本对立，如：自由意志 vs 绝对秩序]
*   **驱动力**：[是什么推动了剧情发展？如：资源的枯竭、秘密的泄露]
