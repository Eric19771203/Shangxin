# 角色绑定模板 (Character Binding Template)

## 1. 主角绑定 (Protagonist Integration)
*   **核心困境**：[世界观的核心规则如何直接导致主角的困境？]
    *   *示例：世界禁止半妖，而主角是半妖。*
*   **特殊性**：[主角为何能打破规则？]
    *   *示例：主角拥有失传的上古血脉。*
*   **动机源头**：[世界观的哪一部分（历史/阶级/资源）驱动了主角的行动？]
    *   *示例：为了夺回被财团垄断的救命药剂。*
*   **成长弧光**：[主角对世界的认知如何变化？]
    *   *示例：从只想生存到想要改变规则。*

## 2. 反派绑定 (Antagonist Integration)
*   **核心主张**：[反派的行为是否符合世界观的逻辑？]
    *   *示例：为了维持秩序必须牺牲少数人。*
*   **资源掌控**：[反派掌握了世界的哪些核心资源？]
    *   *示例：控制了所有的净水水源。*
*   **对立根源**：[反派与主角的冲突是否源于世界观的根本矛盾？]
    *   *示例：秩序 vs 自由，垄断 vs 共享。*

## 3. 配角/众生相 (Supporting Characters)
*   **受害者**：[谁是被世界观规则压迫的典型？]
*   **维护者**：[谁是既得利益者的典型？]
*   **旁观者**：[普通人如何在夹缝中生存？]

## 4. 剧情驱动 (Plot Driver)
*   **开端**：[什么事件打破了主角与世界的平衡？]
*   **中段**：[主角如何探索世界的边界与真相？]
*   **高潮**：[最终决战是否动摇了世界的核心规则？]
