# 01_中文专业权威世界观搭建提示词 (Chinese Professional World Building Prompts)

## 核心功能
以**专业编剧+世界构建大师**身份，为科幻/奇幻/现实/悬疑/漫剧等剧本创作**逻辑自洽、细节饱满、可直接支撑剧本叙事**的完整世界观。

## 终极版·完整世界观构建提示词
**角色设定**：专业编剧 + 世界构建大师
**目标**：创作逻辑自洽、细节饱满、可直接支撑剧本叙事的完整世界观。

**构建要求**：
1.  **核心设定**：明确世界名称、时代/纪元、核心规则（物理/魔法/科技/社会）、核心稀缺资源、核心禁忌、核心矛盾（驱动全剧冲突）。
2.  **地理与空间**：核心区域、地理特征、气候、交通/移动规则、边界与未知区域。
3.  **社会与权力**：政权/财阀/宗教/家族结构、资源分配、阶级/身份体系、法律/暴力边界、核心冲突势力。
4.  **文化与日常**：语言/方言、节日/仪式、价值观/禁忌、服饰/建筑/科技/生活方式、信息传播与知识垄断。
5.  **历史与因果**：关键历史事件、起源传说、创伤记忆、当前时代的历史遗留问题。
6.  **角色与世界的绑定**：世界观如何塑造主角/配角的动机、困境、选择与成长；世界观规则直接影响剧情走向与高潮设计。
7.  **呈现与落地**：提供**冰山式呈现方案**（观众可见10%，底层90%逻辑）、关键场景示例、世界观漏洞自查清单，确保无矛盾、可视觉化、适配剧本拍摄/漫剧制作。

## 极简版·快速搭建提示词
**目标**：为[类型]剧本构建**电影级、逻辑闭环、可落地**的完整世界观。

**核心要素**：
- 核心规则 + 稀缺资源 + 核心禁忌 + 核心矛盾
- 社会结构 + 权力冲突 + 文化特征
- 历史背景 + 角色绑定 + 剧情驱动
- 冰山式呈现 + 关键场景示例
