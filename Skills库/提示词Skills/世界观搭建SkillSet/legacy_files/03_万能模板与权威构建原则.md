# 03_万能模板与权威构建原则 (Universal Templates & Principles)

## 核心功能
提供可直接套用的中英文万能模板，以及构建世界观的四大权威原则，确保逻辑闭环和可落地性。

## 中文模板
> 为[剧本类型]创作**电影级、逻辑闭环、可落地**的完整世界观：
> 1. **核心**：世界名[XX]、时代[XX]、核心规则[XX]、稀缺资源[XX]、禁忌[XX]、核心矛盾[XX]
> 2. **地理**：核心区域[XX]、气候[XX]、交通[XX]、边界[XX]
> 3. **社会**：权力结构[XX]、阶级[XX]、法律/暴力[XX]、势力[XX]
> 4. **文化**：语言/仪式/价值观/科技/建筑[XX]
> 5. **历史**：关键事件[XX]、起源/创伤[XX]
> 6. **角色绑定**：世界观如何塑造主角动机/困境/选择
> 7. **呈现**：冰山式展示 + 关键场景 + 自查清单

## 英文模板
```markdown
Create a cinematic world for a [genre] script:
1. **Core**: Name [XX], Era [XX], Rules [XX], Scarce Resource [XX], Taboo [XX], Core Conflict [XX]
2. **Geography**: Regions [XX], Climate [XX], Transport [XX], Boundaries [XX]
3. **Society**: Power [XX], Class [XX], Law/Violence [XX], Factions [XX]
4. **Culture**: Language/Rituals/Values/Tech/Architecture [XX]
5. **History**: Key Events [XX], Origins/Traumas [XX]
6. **Character**: How world shapes protagonist arc & plot
7. **Presentation**: Iceberg principle + key scenes + consistency check
```

## 权威构建原则（必看）
1.  **冰山法则**：观众只看10%，你必须构建90%底层逻辑，确保无矛盾。
2.  **冲突驱动**：世界观所有设定都要服务于**核心矛盾**，不写无用设定。
3.  **角色绑定**：世界观直接塑造角色的**动机、困境、选择**，是剧情的发动机。
4.  **可视觉化**：所有设定要能转化为**画面、场景、道具、服化道**，适配拍摄/漫剧。
