# 02_英文AI生成专用世界观提示词 (English AI World Building Prompts)

## 核心功能
专为AI生成设计，包含完整版和精简版Prompt，适用于ChatGPT、Claude等大语言模型生成高质量世界观。

## 完整版·AI世界构建Prompt
```markdown
Act as a professional screenwriter and worldbuilding master with 20+ years of experience in film, TV, and comic production.

Create a **fully realized, logically consistent, cinematic world** for a [genre: sci-fi/fantasy/realistic/mystery/anime comic] script/story.

Include:
1. **Core World Concept**: Name, era, central rules (physics/magic/tech/society), scarce resource, absolute taboo, core conflict that drives the entire narrative.
2. **Geography & Space**: Key regions, climate, transportation/movement rules, boundaries & unknown zones.
3. **Society & Power**: Political/religious/corporate structures, resource distribution, class system, laws & violence limits, major factions in conflict.
4. **Culture & Daily Life**: Language, rituals, values, fashion, architecture, tech, information flow, knowledge hierarchy.
5. **History & Lore**: Founding myths, pivotal historical events, traumas, legacy shaping the present.
6. **Character Integration**: How the world shapes protagonist/antagonist motivations, choices, arcs; how world rules drive plot beats and climax.
7. **Iceberg Presentation**: 10% visible to audience, 90% underlying logic; key scene examples; worldbuilding consistency checklist.

Ensure the world is **cinematic, visually striking, plot-driven, and ready for screen adaptation/comic production**.
```

## 精简版·快速生成Prompt
```markdown
Create a cinematic, logically consistent world for a [genre] screenplay.

Include:
- Core rules, scarce resource, taboo, conflict
- Society, culture, history
- How it drives character and plot

Use iceberg principle: show 10%, build 90% logic.
Make it ready for film/comic production.
```
