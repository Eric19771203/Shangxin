# 07_漫剧角色定稿官方设定集 (Character Design Bible)

## 核心功能
严格匹配漫剧连载、电影级3D制作、AI生成的专业标准，和世界观体系完全打通，确保人设不崩、细节可落地。

## 一、角色定稿核心
1.  **核心锚点**：姓名、定位、一句话人设（绑定矛盾）、阶级站位、核心作用。
2.  **基础档案**：性别/年龄/身高、阵营、核心标签、口头禅。
3.  **外貌细节**：面部五官（辨识度核心）、皮肤/毛发/体态质感（电影级）。
4.  **性格弧光**：表层/深层性格、执念动机、弱点、成长弧光。
5.  **能力规则**：核心能力、限制反噬、武器道具、成长线。
6.  **服化道**：配色体系、分场景造型（日常/战斗/特殊）、标志性配饰。

## 三、避坑指南
1.  **辨识度统一**：疤痕/发型/瞳色等记忆点绝不更改。
2.  **能力守则**：绝不打破世界观规则，无“机械降神”。
3.  **行为逻辑**：选择符合人设，拒绝OOC。
4.  **视觉配色**：主色调固定，强化记忆。
5.  **细节有效**：所有装饰服务于人设剧情。
