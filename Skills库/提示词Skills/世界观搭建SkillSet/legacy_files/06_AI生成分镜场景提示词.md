# 06_AI生成分镜场景提示词 (AI Storyboard & Scene Prompts)

## 核心功能
专为AI绘图工具（Midjourney/SD/ComfyUI）设计，提供中英文场景生成提示词，确保出图符合漫剧分镜标准。

## 英文·AI场景/概念图生成提示词
```markdown
masterpiece, best quality, 8K, ultra realistic, photorealistic, cinematic lighting, movie still,
3D cinematic concept art, worldbuilding for [填写漫剧类型] manhua,
[填写核心场景，示例：cyberpunk vertical city, upper glass dome headquarters, middle holographic commercial district, lower slum with pipes],
ultra detailed, intricate details, sharp focus, hyper realistic, octane render,
dramatic lighting, wide angle composition, depth of field,
no blurs, no flaws, no distortions, professional concept art for manhua production
```

## 中文·AI分镜/场景生成提示词
```markdown
电影级漫剧世界观分镜概念图，超拟真3D建模质感，8K超清，极致细节，
[填写场景名称，示例：异能都市地下黑市]，
场景细节：[填写场景特征，示例：昏暗的地下空间，管道纵横，随处可见非法异能交易，墙面画满禁忌异能涂鸦，暗处有安保人员巡逻，霓虹灯光斑驳，氛围紧张压抑]，
电影级光影，超广角构图，景深拉满，氛围到位，无噪点，无崩坏，适配漫剧正片分镜标准
```

## 中文·AI剧情/世界观生成提示词
```markdown
以专业影视编剧+漫剧世界观构建大师的身份，为[填写漫剧类型]漫剧，创作一套逻辑闭环、细节饱满、可直接落地连载的完整世界观。
要求：
1. 严格遵循冰山法则，90%底层逻辑支撑，10%内容呈现，适配漫剧的视觉化叙事
2. 包含核心定调、基础规则、地理场景、权力体系、文化细节、历史背景、角色绑定、核心矛盾8个部分
3. 所有设定都服务于剧情和角色，无无效自嗨内容，核心规则全程统一，无逻辑bug
4. 适配电影级超拟真3D漫剧的画面呈现，所有场景、设定都能直接转化为分镜
5. 预留连载伏笔和拓展空间，核心矛盾清晰，有强戏剧冲突，能支撑至少3季的主线剧情
```
