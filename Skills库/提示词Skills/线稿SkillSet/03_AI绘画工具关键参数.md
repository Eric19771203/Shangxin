# 03_AI绘画工具关键参数 (AI Tool Key Parameters)

## 核心功能
提供在使用SD/ComfyUI等工具生成高质量线稿时，必须配置的ControlNet关键参数设置。

## 关键配置
仅靠描述不够，必须搭配**ControlNet+预处理器**才能稳定出高质量线稿：

- **控制类型**：Lineart（线稿）
- **预处理器（按场景选）**：
  - 写实/照片：`lineart_realistic`
  - 动漫/二次元：`lineart_anime` / `lineart_anime_denoise`
  - 通用/场景：`lineart_standard`
- **模型**：`control_v11p_sd15_lineart`
- **控制权重**：`0.8–1.0`（越高越贴合原图）
- **重绘幅度**：`0.6–0.8`（保留结构+优化线条）
- **采样器**：`Euler a` / `DPM++ 2M Karras`，步数20–30
