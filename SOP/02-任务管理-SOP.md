# 任务管理 SOP

**生效日期**：2026年2月28日
**版本**：v1.0

---

## 📋 任务生命周期

### 标准任务流程

```
1. 任务创建
    ↓
2. 任务分析和拆分
    ↓
3. 任务分配
    ↓
4. 任务执行
    ↓
5. 进度汇报
    ↓
6. 任务审核
    ↓
7. 任务交付
    ↓
8. 任务归档
```

---

## 🎯 任务优先级

### P0 - 紧急且重要
- 用户立即需要的
- 阻塞其他任务的
- **响应时间**：立即执行

### P1 - 重要但不紧急
- 重要项目的核心任务
- 有明确截止日期的
- **响应时间**：24 小时内

### P2 - 不紧急但重要
- 长期优化项目
- 能力建设
- **响应时间**：72 小时内

### P3 - 不紧急也不重要
- 锦上添花的任务
- 有空再做
- **响应时间**：有空就做

## 📊 任务状态

| 状态 | 说明 |
|------|------|
| 📝 **待办** | 已创建，未分配 |
| 📋 **已分配** | 已分配给负责人 |
| 🔄 **进行中** | 正在执行 |
| ⏸️ **阻塞** | 被阻塞，等待外部条件 |
| ✅ **已完成** | 已完成，待审核 |
| 🎉 **已交付** | 已交付给用户 |
| ❌ **已取消** | 已取消 |

## 📝 任务模板

### 任务创建模板

```markdown
# 任务标题

## 任务描述
（清晰描述任务内容和目标）

## 优先级
P0 / P1 / P2 / P3

## 截止日期
YYYY-MM-DD

## 交付物
（列出需要交付的具体内容）

## 相关信息
（任何相关的链接、文档、背景信息）
```

## ⏰ 汇报机制

### 实时汇报
- **P0 任务** - 每 2 小时汇报一次进度
- **P1 任务** - 每天汇报一次进度
- **P2/P3 任务** - 阶段性汇报

### 每日汇报
- 每天早上 6:00 - 昨日任务成果报告（已设置定时任务）
- 包括：已完成、进行中、遇到的问题

### 阶段性汇报
- 项目里程碑时汇报
- 交付前汇报
